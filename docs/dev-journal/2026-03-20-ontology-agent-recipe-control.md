## 사용자 요청 요약

MASON 리서치에 이어, 범위를 좁혀서 구체적인 시스템 구현 가능성 조사. "온톨로지 기반 불량 분석 → 레시피 생성 → 설비 전달" 폐쇄 루프 시스템을 지식그래프 + LLM 멀티에이전트로 구현할 수 있는지, 아키텍처와 제조 현실 양면에서 조사.

---

## 문제 정의

1. **기술적 실현 가능성**: 온톨로지/KG + LLM 멀티에이전트로 이 시스템을 구현할 수 있는가?
2. **제조 현실성**: 기존 APC/FDC/SPC 대비 어떤 이점이 있고, 안전/책임 이슈는?
3. **방향 제시**: 구현한다면 어떤 아키텍처와 단계적 접근이 적절한가?

---

## 핵심 가정

1. 사용자는 MASON 리서치를 통해 온톨로지 기본 개념을 이해하고 있다
2. 실제 구현을 목표로 하므로, 이론보다 실현 가능성과 구체적 방향이 중요
3. Human-in-the-loop은 필수 (완전 자율 제어는 아님)
