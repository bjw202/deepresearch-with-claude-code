## 세션 메타

- **날짜**: 2026-03-24
- **주제**: 레이저 소스/광학계/하드웨어 용어·개념·원리 종합 정리
- **참여 에이전트**: Researcher 3명 + Critic 1명 + Synthesizer 1명 + Journal 1명
- **리서치 폴더**: `docs/research/2026-03-24-laser-fundamentals-guide/`

## 사용자 요청 요약

절삭가공 출신 엔지니어가 레이저 전문인력과 대등하게 토론할 수 있도록, 레이저 소스/광학계/하드웨어의 용어·개념·원리를 총망라하여 정리. 단순 용어 사전이 아니라 "왜 이것이 중요한가"까지 연결하는 실무 참고서 수준 요구.

---

## 문제 정의

- 사용자는 레이저 가공/프로세싱 업무를 하고 있으나 레이저 기초지식(소스/광학계/하드웨어) 부족
- 공정 최적화/설계 역량은 있으나 레이저 전문인력과의 토론에서 한계 발생
- 목표: 절삭가공 경험을 브릿지로 활용하여 레이저 개념을 체계적으로 정리한 참고 가이드 작성

---

## 핵심 가정

- 독자의 물리적 직관은 엔지니어 수준 (파동, 에너지 개념 이해 가능)
- 절삭가공 경험(공구-재료 상호작용, 열, 절삭 파라미터)을 레이저 개념 이해의 브릿지로 활용 가능
- 용어 사전이 아닌 "왜 이것이 중요한가"까지 연결하는 실무 참고서 수준 필요
- 레이저 공학의 정량적 수치(M², BPP, 열렌즈 효과 등)는 검색으로 최신/정확 데이터 확보 필요

---

## Question Expansion

1. **핵심 질문**: 레이저 소스/광학계/하드웨어의 용어·개념·원리를 절삭가공 엔지니어가 전문인력과 토론 가능한 수준으로 정리
2. **검증 필요 전제**:
   - 물리적 직관은 있으나 정량적 관계(M², BPP, 열렌즈 등)가 부족하다는 가정
   - 절삭가공 브릿지가 실제로 레이저 이해에 유효한가
3. **인접 질문**:
   - 레이저-물질 상호작용의 기초 메커니즘 (절삭가공과 어떻게 다른가)
   - 빔 품질 평가 방법론 (M², BPP 측정/해석 실무)
   - 레이저 안전 등급과 운용 주의사항
4. **반대 시나리오**: 용어 정리만으로는 부족하며, 의사결정 로직(파라미터 선택 → 공정 결과 예측)까지 연결해야 실전 토론 가능
5. **이질 도메인 유추**: 절삭가공 공구-재료 상호작용 → 레이저-재료 상호작용 매핑 가능. 절삭 깊이/이송속도/절삭열 → 플루언스/스캔속도/열 영향부(HAZ)의 구조적 유사성 활용 가능

## 의사결정 로그

### Researcher 3명 구성 결정

- **결정**: 소스/광학계/하드웨어로 분리
- **근거**: 레이저 시스템의 물리적 구성요소가 자연스럽게 3개 도메인으로 분리됨. 각 도메인의 전문성이 명확히 구분되어 분업 효과가 큼
- **대안**: Researcher 4명 (레이저-물질 상호작용 별도) → 하드웨어와 분리하면 공정 파라미터 맥락 단절 우려로 기각

### 절삭가공 브릿지 배치

- **결정**: Researcher 1 (레이저 물리 기초) 보고서에서 절삭가공 브릿지 개념 도입
- **근거**: 독자가 가장 먼저 접하는 보고서에서 개념적 기반을 마련해야 이후 보고서 이해가 수월함

### 레이저-물질 상호작용 위치

- **결정**: Researcher 3 (하드웨어/시스템)과 통합
- **근거**: 공정 파라미터(파워/속도/주파수)와 물질 상호작용(흡수율/HAZ/에블레이션 임계값)이 밀접하게 연결됨. 분리 시 "왜 이 파라미터인가" 설명 불가

### 검색 필요성 판단

- **결정**: 검색 수행
- **근거**: 레이저 공학은 정량적 수치가 중요 (예: 파이버 레이저 M² &lt; 1.1, CO₂ 파장 10.6µm, BPP 단위 mm·mrad). 모델 지식만으로는 최신 제품 스펙 및 정확한 수치 신뢰도 부족
