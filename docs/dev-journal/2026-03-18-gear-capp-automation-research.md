# 저널: 기어 가공 CAPP 자동화 심층 리서치

## 세션 메타
- **날짜**: 2026-03-18
- **요청자**: 사용자
- **목적**: 기어 가공 CAPP 자동화, 공구 선정, 공정 최적화 현황 및 AI/LLM 적용 가능성 심층 조사

## 사용자 요청 요약
기어 가공의 공정계획 자동화(CAPP), 공구 선정, 공정 최적화의 현황과 AI/LLM 적용 가능성을 10개 항목에 걸쳐 심층 조사. 결과를 `docs/research/gear-capp-automation.md`에 저장.

## 문제 정의
- 기어 가공은 공정 체인이 복잡하고 도메인 전문 지식 의존도가 높음
- CAPP 자동화는 수십 년 연구 역사가 있으나 기어 특화 연구는 제한적
- AI/ML, 특히 LLM의 등장으로 새로운 가능성이 열리고 있으나 기어 도메인 적용은 초기 단계

## 핵심 가정
- 기어 가공 CAPP는 일반 CAPP보다 도메인 특화 지식 비중이 높음
- LLM 단독으로는 수치적 정확성 보장 불가, 하이브리드 접근 필수
- 상용 SW(KISSsoft, GEMS 등)와의 보완 관계가 현실적 방향

## 의사결정 로그
1. **검색 전략**: Perplexity sonar-pro 5건 + Tavily advanced 4건 병렬 실행 → 비용 효율적
2. **기존 리서치 재활용**: `docs/research/2026-03-18-gear-system/` 4개 보고서의 기존 내용과 중복 최소화
3. **보고서 구조**: 10개 조사 항목 순서대로 + 종합 분석 섹션 추가

## 이슈 / 해결
- **이슈**: 기어 CAPP 특화 LLM 연구가 거의 없음 → 일반 CAPP + 기어 도메인 지식을 교차 분석
- **해결**: 일반 CAPP AI 연구 + 기어 제조 실무 지식을 결합하여 적용 가능 시나리오 도출
