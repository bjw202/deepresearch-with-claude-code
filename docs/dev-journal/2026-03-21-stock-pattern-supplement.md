## 사용자 요청 요약

기존 리서치가 "ML/RL 자동매매 시스템" 프레임으로 작성되어 있었으나, 실제 목적은:
1. 고확률 피봇 포인트 발견 (O'Neil/Minervini/Ryan/쿨라메기/댄쟁거/TraderLion)
2. 패턴 정량화 & 분류 (VCP, CWH, HTF, W, Low Cheat, Episodic Pivot)
3. 차트북(성공/실패 사례집) 자동 생성
4. 유사 패턴 매칭 (현재 차트 → 과거 유사 사례 + 확률)
5. 전처리/정규화: 종목 간 가격 차이, 거래량 의미 차이, 시계열 컨텍스트
6. **자동매매가 아닌 의사결정 보조 도구**

---

## 문제 정의

- 기존 리서치의 RL(Exit 최적화, Position Sizing), 자동 실행 최적화는 목적에서 벗어남
- 패턴 정량화, 차트북, 유사도 매칭, 전처리 관련 내용이 기존에 없었음
- 패턴 성공 기준: +8%/10일 → **+30%/10일로 변경** (사용자: "시원하게 뚫느냐가 관점")

---

## 핵심 가정

1. 2,500종목 x 2년 DB가 충분한 패턴 사례를 제공할 것 (검증 필요)
2. +30%/10일 기준으로 양성 라벨이 극소수일 수 있음 (클래스 불균형 심화)
3. Minervini/O'Neil의 미국 기반 수치가 한국 시장에 교정 가능 (검증 필요)
4. 패턴 유사도가 향후 성과 예측에 유의미한 정보를 제공 (검증 필요)
