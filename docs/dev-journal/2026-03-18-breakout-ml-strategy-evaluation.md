# 세션 저널: 돌파 트레이딩 ML 전략 평가

## 세션 메타
- 날짜: 2026-03-18
- 유형: 리서치 평가
- 산출물: `docs/research/market-perspective-evaluation.md`

## 사용자 요청 요약
스윙트레이더가 CWH/VCP/HTF 돌파 패턴을 ML로 개선하려 함. ChatGPT가 4단 구조(표현학습->확률예측->랭킹->실행최적화)를 제안. 이에 대한 6개 관점 실증 평가 요청.

## 문제 정의
- 돌파 패턴이 실제로 양의 기대값을 가지는가?
- ML이 인간 트레이더를 대체/보완할 수 있는가?
- 제안된 구조와 피처가 실전에 적합한가?

## 핵심 가정
- 사용자는 미국 주식시장 스윙 트레이더
- 기존 패턴 인식 능력은 보유하나 미스율 개선 원함
- 자동화보다는 의사결정 보조 목적

## 의사결정 로그
1. **검색 전략**: Tavily advanced + Bulkowski 직접 데이터 추출 조합. Perplexity는 Bash 제한으로 사용 불가, Tavily MCP로 대체
2. **구조**: 6개 리서치 질문을 병렬 검색으로 처리 후 통합
3. **핵심 발견**: 섹터 필터가 VCP 승률에 결정적 영향 (51.3% vs 유의미하게 높음)
4. **추가 권장**: 0단(레짐 판단) 추가, 피처 7개 추가 제안

## 이슈 / 해결
- Bash 권한 거부 -> search.sh 대신 Tavily MCP 도구 직접 사용
- VCP 독립 학술 백테스트 부재 -> Minervini 감사 실적 + Finer Market Points 독립 분석으로 보완
