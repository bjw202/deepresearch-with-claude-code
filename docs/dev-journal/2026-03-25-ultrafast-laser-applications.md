## 사용자 요청 요약
펨토초/피코초 초단파 레이저 어플리케이션을 여러 분야에 걸쳐 종합 조사

## 문제 정의
- 초단파(USP) 레이저의 다분야 어플리케이션 지도를 그리는 것
- 산업 제조, 의료/바이오, 과학, 신흥 분야를 포괄
- 펨토 vs 피코 선택 기준, 시장 트렌드, 기술 방향성 포함

## 핵심 가정
- 사용자가 레이저 가공 실무자이므로 산업 제조 관점이 가장 실용적
- 의료/과학 분야는 시야 확장 및 기술 이해도 심화 목적
- 시장/트렌드 정보는 업무 방향성 판단에 활용

## Question Expansion
1. 핵심 질문: 펨토/피코 레이저의 다분야 어플리케이션과 기술적 특성
2. 검증 필요 전제:
   - "USP = HAZ 제로"가 모든 조건에서 유효한가
   - 펨토가 항상 피코보다 우월한가
3. 인접 질문:
   - 산업 채택 장벽(비용/신뢰성)은 어디까지 해소되었는가
   - 버스트 모드/GHz 반복률의 영향
   - 나노초/CW 대비 경제적 정당성 임계점
4. 반대 시나리오: 비용 하락에도 기존 공정 우위 영역이 커서 시장 성장 제한적
5. 이질 도메인 유추: EUV 리소그래피 전환과의 유사성

## 이슈 / 해결
- **search.sh bash 호환 문제**: 환경에 bash가 없어 서브에이전트 검색 전부 실패. shebang을 `#!/bin/zsh`로, `BASH_SOURCE[0]`를 `${BASH_SOURCE[0]:-$0}`로 수정하여 해결.
- **서브에이전트 백그라운드 검색 실패**: 서브에이전트가 검색을 `run_in_background`로 실행 → 별도 bash 프로세스에서 실패. 프롬프트에 "포그라운드에서 순차 실행" 명시하여 해결.
- **API 한도 1차 실패**: 첫 발사 시 서브에이전트 전원 API rate limit. 리셋 후 재발사로 해결.
- **Critic 품질 B+**: Repair Pass 불필요 판정. HAZ 수치 불일치, 제조사 출처 편향, 뇌전증 확신도 과대를 synthesis에 반영.
