## 사용자 요청 요약

"openclaw가 AI 신에서 핫한데, 쓸모가 있는지 의문. 조사해달라. 사용사례, 발전 가능성 포함. 쉬운 언어로."

- 핵심 관심: 실제 유용성, 사용사례, 미래 발전 가능성
- 톤: 전문 용어 최소화, 쉬운 설명 요구

---

## 문제 정의

- OpenClaw는 오픈소스 로컬 AI 에이전트 프레임워크
- PC에서 AI가 브라우저 조작, 파일 관리, 셸 명령 등을 자동화
- 사용자는 "핫하다"는 주장에 대해 실제 근거를 원하며, 비판적 시각도 요구
- 단순 소개가 아닌 "쓸모 판단"이 목적

---

## 핵심 가정

1. 사용자는 개발자이지만 AI 에이전트 프레임워크에는 초보
2. "핫하다"는 주장의 실제 근거(GitHub 스타, 커뮤니티 활동 등) 검증 필요
3. 장점만 나열하면 안 됨 — 비판적 관점(보안, 세팅 비용, 한계) 필수
4. 기존 도구(RPA 등)와의 차별점이 명확해야 쓸모 판단 가능

| 3 | 인접 질문 | 유사 도구(Open Interpreter, AutoGPT 등)와 비교 시 장단점? 보안 위험은? 비개발자도 쓸 수 있는가? |
