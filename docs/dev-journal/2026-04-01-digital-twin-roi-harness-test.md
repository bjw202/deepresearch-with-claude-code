## 문제 정의

제조업에서 디지털 트윈 기술의 실제 도입 현황, ROI, 실무자 경험을 3개 채널(웹/학술/커뮤니티)로 독립 조사하여 교차 검증하고, 최종적으로 통합된 판단을 도출한다.

---

## 핵심 가정

1. **Type B (기술 동향/전략)** — web, academic, community 3채널 모두 유의미한 정보를 제공할 수 있다.
2. **검색 시간 범위: `--days 365`** — 디지털 트윈은 고속 변화 기술로 최근 1년 데이터가 핵심이다.
3. **성능 테스트 목적** — 리서치 범위를 과도하게 확대하지 않고 하네스 성능 평가에 적합한 수준으로 한정한다.

## Question Expansion

**핵심 질문:** 제조업에서 디지털 트윈의 실제 ROI는 얼마이며, 도입 현황은 어느 단계인가?

**전제 검증 필요:**
1. "제조업 전반에 광범위 도입" 전제 — 실제로는 대기업/특정 산업 편중일 가능성
2. ROI 수치의 독립성 — 벤더가 발표한 수치와 독립 연구 수치가 다를 수 있음

**인접 질문:**
1. 도입 실패 사례는 어떤 패턴인가? (실패 요인 역분석)
2. 중소기업과 대기업 간 도입 격차 — 비용/기술 장벽 차이
3. 디지털 트윈과 MES/PLM 기존 시스템의 통합 관계

**반대 시나리오:** ROI 수치 과대평가 — 벤더 주도 케이스 스터디에서 성공 편향이 심하여 실제 산업 평균은 훨씬 낮을 가능성

**이질 도메인 유추:** 의료 분야의 환자 디지털 트윈 — 동일한 "실물 → 가상 미러링" 구조에서 데이터 품질과 모델 업데이트 주기가 ROI를 결정하는 핵심 변수로 작용함. 제조업에서도 센서 데이터 품질과 모델 갱신 빈도가 ROI의 숨은 결정 변수일 수 있음.
