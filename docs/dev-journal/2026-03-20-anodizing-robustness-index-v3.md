---

## 사용자 요청 요약

아노다이징 피막의 강건성(내식성, 내구성)을 객관적으로 평가하고, 단일 숫자 지수로 정량화하여 제조 현장의 출하/공정 판정 기준으로 활용할 수 있는 방법론 리서치. 6개 핵심 질문(인증체계, EIS 지표, EIS 실용성, 비EIS 결합, 환경 특화, 지수 설계).

---

## 문제 정의

- **핵심**: 다항목 품질 시험 결과를 단일 강건성 지수로 통합하여 현장 판정 기준 제공
- **전제 검증 필요**:
  1. EIS의 아노다이징 피막 신뢰성 (유기코팅 차용 아닌지)
  2. 단일 지수화의 실용적 우위
- **반대 시나리오**: 단일 지수의 "평균의 함정" — Gate + Score 2단계로 완화 가능

---

## 핵심 가정

1. EIS가 비파괴 정량 평가의 핵심 도구가 될 수 있다
2. 기존 Go/No-Go 체계를 넘어 종합 점수화가 제조 현장에 가치를 준다
3. 동남아 고온고습 + 인체 땀 환경이 주요 사용 환경이다

## 의사결정 로그 (계속)

| 시점 | 결정 | 근거 |
|------|------|------|
| R1 완료 | Tupy SQI를 ARI 구조적 참고로 채택 | Gate+Score 하이브리드의 가장 직접적 선례 |
| R2 완료 | EIS를 일상 QC가 아닌 Tier 3(자격 인증)으로 배치 | ISO 2931 어드미턴스가 봉공 QC에 충분, EIS 인라인 미해결 과제 다수 |
| R3 완료 | 3-Tier 시험 체계 채택 | 전수 비파괴 + 로트 샘플링 + 자격 인증의 현실적 계층 |
| Critic | EIS |Z| 임계값 통일 보류 | R2(10⁶)와 R3(10⁵)의 3오더 차이는 아노다이징 직접 검증 데이터 부재에서 기인. 검증 전 어떤 수치도 확정하지 않음 |
| Critic | 가중치를 "검증 전 초안"으로 명시 | Critic 지적: 표 형태가 검증된 수치 인상을 줌 |
| 통합 | ARI를 인증 대체가 아닌 보완으로 위치 설정 | Critic 지적: 규제 준수와의 관계 명확화 필요 |
