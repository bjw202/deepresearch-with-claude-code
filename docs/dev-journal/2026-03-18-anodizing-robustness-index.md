# 세션 저널: 아노다이징 피막 강건성 지수화 방법 리서치

- 날짜: 2026-03-18
- 유형: 딥리서치
- 주제: 스마트폰 메탈 프레임 아노다이징 피막 강건성의 정량적 지수화 방법
- 참여 에이전트: 메인(오케스트레이터), eis-researcher, index-researcher, alt-researcher, journal

## 사용자 요청 요약

스마트폰 메탈 프레임의 아노다이징(양극산화) 피막 강건성을 정량적 지수로 평가하는 방법을 조사해 달라는 요청. 현재 EIS(전기화학 임피던스 분광법)를 사용 중이나 그래프 해석에 주관성이 있어, 이를 단일/소수의 숫자 지표로 체계화하여 제조 현장에서 pass/fail 판정에 활용하고자 함.

### 배경 컨텍스트

- 아노다이징 피막: 알루미늄 표면에 산화알루미늄(Al2O3) 보호막을 전기화학적으로 형성하는 공정
- 용도: 스마트폰 프레임의 디자인(착색)과 내스크래치성 확보
- 핵심 리스크: 동남아 등 고온고습 환경에서 땀 속 염분이 피막 결함부로 침투하여 박리 유발
- 트레이드오프: 두꺼운 피막은 보호력이 높지만 열팽창 차이로 크랙 발생, 컬러링 품질 저하 가능

## 문제 정의

1. **해석 주관성 해소**: EIS 나이퀴스트/보드 플롯의 정성적 해석을 정량 지표로 대체
2. **지수화(Indexing)**: 피막 강건성을 단일 또는 소수의 숫자 지표로 압축
3. **현장 적용성**: 제조 라인에서 pass/fail 판정 기준으로 활용 가능한 체계 구축

## 핵심 가정

- EIS 장비와 데이터 인프라는 이미 구축되어 있으므로, EIS 기반 지수화가 가장 실용적인 출발점
- 완전히 새로운 측정법 도입보다 기존 데이터의 해석 체계화가 우선
- 주요 스트레스 조건: 고온고습(85도C/85%RH) + 인공 땀(NaCl 기반 염분)
- 피막 두께 범위: 스마트폰 프레임 표준인 5\~25um 수준

## 의사결정 로그

### 1. 리서치 범위 결정

- **결정**: EIS 기반 방법론을 중심에 두되, 대안 평가법과 복합 지수 가능성도 병행 조사
- **이유**: 사용자가 이미 EIS를 운용 중이므로 전환 비용 최소화가 중요. 그러나 EIS 단독으로는 피막의 모든 측면(기계적, 화학적, 구조적)을 반영하기 어려울 수 있음

### 2. 서브에이전트 구조 설계

- **결정**: 3개 리서처로 관점 분리
  - `eis-researcher`: EIS 등가회로 모델링, 파라미터 추출, 지수화 기존 방법론 심층 조사
  - `index-researcher`: 코팅/피막 강건성 지수화 사례 전반 조사 (PCI, Coating Degradation Index 등)
  - `alt-researcher`: EIS 외 평가법(CASS, 솔트스프레이, 마이크로하드니스, SEM 기공률 등) 및 복합 지수 구축 가능성
- **이유**: EIS 파라미터 해석, 지수화 프레임워크, 대안/보완 측정법은 각각 독립적 조사 영역이며 병렬화 이점이 큼

### 3. 관점 분리의 근거

- EIS는 전기화학적 관점(이온 전도, 전하 이동)에 강하지만, 기계적 강건성(경도, 내마모)이나 구조적 특성(기공 구조, 봉공 품질)은 별도 측정이 필요할 수 있음
- 최종적으로 복합 지수를 구성할 때 어떤 파라미터를 조합할지는 세 관점의 결과를 통합해야 판단 가능

## 이슈 / 해결

- **이슈**: EIS 기반 지표 대부분이 유기 코팅(페인트, 에폭시) 연구에서 도출. 아노다이징(무기 산화물) 피막에 직접 적용 시 등급 경계값의 적절성 미검증
  - **해결 방향**: ARI Phase 2에서 아노다이징 시편 전용 회귀 분석으로 경계값 교정 필요
- **이슈**: 시간 적분 지표 ∫(1/|Z|)dt가 최고 성능이나 연속 모니터링이 필요하여 출하 전 1회 검사에는 부적합
  - **해결 방향**: 출하 전에는 |Z|₀.₁Hz 단일 시점, 공정 관리에는 CDI% 변화 추적 활용

## Rejected Alternatives

### 서브에이전트 구조 대안

- **단일 에이전트 통합 조사**: 하나의 에이전트가 EIS + 대안 + 지수화를 모두 조사하는 방식. 각 영역의 깊이가 얕아질 우려로 기각.
- **5개 에이전트 세분화** (EIS이론, EIS실무, 기계적평가, 화학적평가, 지수화): 분업 실익 대비 조율 비용이 높다고 판단. 3개로 충분.

### 접근 방식 대안

- **ML 기반 EIS 패턴 분류부터 시작**: 딥러닝으로 EIS 스펙트럼을 직접 분류하는 방법. 해석 가능성이 낮고, 학습 데이터 확보 문제가 있어 1차 접근에서는 제외. 다만 후속 연구로는 유효.
- **EIS를 버리고 다른 비파괴 검사법으로 전환**: 사용자의 기존 인프라 활용이 불가하므로 기각. 보완적 병용은 alt-researcher가 조사.
