# 레이저 싱귤레이션 흄/파티클 2차 오염 제어: 종합 보고서

## Executive Summary

몰딩된 기판 패키지 어레이(Cu + EMC + FR-4)의 피코초(ps) 레이저 절단에서 흄/파티클 2차 오염을 제어하기 위한 종합 전략을 도출하였다. 3명의 Researcher가 각각 레이저 파라미터 최적화, 배출/제거 시스템, 재료별 어블레이션 특성을 조사하였고, Critic이 교차 검증하였다.

### 핵심 결론 5가지

1. **구리와 EMC의 어블레이션 임계값 \~10배 차이가 근본 문제이다.** 단일 파라미터 세트로는 두 재료 모두에서 부산물을 최소화할 수 없으며, 층별 적응형 파라미터 전략이 필요하다.

2. **"발생 억제 + 즉시 배출 + 재증착 방지"의 3중 방어가 최적 전략이다.** 어느 하나만으로는 충분하지 않다.

   - 1선: 레이저 파라미터 최적화(저플루언스 멀티패스, 적절한 펄스폭/파장)
   - 2선: 어시스트 가스 + 로컬 흡입으로 즉시 배출
   - 3선: 보호 필름(HogoMax)으로 재증착 방지 + 후세정

3. **ps 레이저가 생산성과 품질의 최적 균형점이다.** ns 대비 부산물이 현저히 적고, fs 대비 비용/생산성이 유리하다. 단, fs가 품질 최우선 조건에서 HAZ를 극소화하는 장점이 있다.

4. **다층 구조의 재료 인터페이스 전환점이 최대 오염 위험 구간이다.** Cu→EMC 전환 시 혼합 부산물(전도성 + 절연성 + 유기)이 생성되며, 단일 세정법으로 제거 곤란하다.

5. **과도한 어시스트 가스 압력은 역효과를 낳는다.** Mach Stem Disk 형성, 플라즈마 플룸 교란 등으로 오히려 절단 품질이 악화된다.

## 1. 통합 전략: 3중 방어 체계

### 1선: 발생 억제 — 레이저 파라미터 최적화

| 파라미터 | 권장 | 근거 | 확신도 |
| --- | --- | --- | --- |
| **펄스폭** | 1-10 ps | ns 대비 debris 현저 감소, fs 대비 비용/생산성 유리 | 높음 |
| **파장** | 532 nm (Green) 또는 355 nm (UV) | Cu 흡수율 + EMC 광화학 어블레이션 최적화 | 높음 (ns 데이터 기반, ps 검증 필요) |
| **플루언스** | 문턱값의 \~e²배 근처 (Cu: \~0.4 J/cm², EMC: 별도 최적화 필요) | Nolte et al. 이론. 단, 복합재료 검증 없음 | 중간 |
| **패스 전략** | 저에너지 멀티패스 (8-12회) | 2008년 ns 레이저 데이터, ps에서 최적 패스 수는 재검증 필요 | 중간 |
| **빔 프로파일** | Top-hat 권장 | 단일 재료에서 균일 어블레이션 확인, 복합재료 검증 필요 | 중간 |
| **반복률** | 100-500 kHz | 열축적 최소화 + 합리적 생산성 | 높음 |

**고급 전략** (장비 지원 시):

- **MHz 버스트 모드**: 총 에너지를 서브펄스로 분배하여 대형 파티클 형성 억제. 단, 현재 데이터는 fs 레이저 + 단일 금속 기반이므로 ps + 복합재료에서 정량적 검증 필요.
- **층별 적응형 파라미터**: Cu층(높은 플루언스) → EMC층(낮은 플루언스) 전환. 인라인 모니터링(LIBS, 광다이오드)으로 재료 전환 감지 시 자동 조정.

### 2선: 즉시 배출 — 어시스트 가스 + 흡입 시스템

| 요소 | 권장 | 주의사항 |
| --- | --- | --- |
| **어시스트 가스** | N2 (순도 99.9%+) | Cu 산화 방지, EMC 탄화물 억제. Ar도 대안이나 비용 높음 |
| **분사 방식** | Side-blow 또는 Cross-jet | Coaxial은 \~90% 면적 차단(choking) 발생 가능 |
| **가스 압력** | MSD 형성 임계 이하 (재료/두께별 실험 결정) | 과도 시 역효과. 금속 판재 기준 7→4 bar 감압 시 품질 개선 사례 있으나, 반도체 패키지 커프 스케일에서의 검증 필요 |
| **로컬 흡입** | 절단점 근방 음압 형성, 가스 튜브 레이저 경로 평행 배치 | 특허 WO2019054945A1 (EMC 기판 직접 대상) |
| **필터** | HEPA + 활성탄 필수 | SiO2 미립자(HEPA) + 유기 VOC(활성탄) 동시 처리 |

### 3선: 재증착 방지 + 후세정

| 단계 | 방법 | 비고 |
| --- | --- | --- |
| **가공 전** | DISCO HogoMax 수용성 보호 필름 코팅 (0.82-1.58 um) | 업계 표준. 금속 불순물 &lt;50ppb, 30초 DI water 세정으로 완전 제거 |
| **가공 후 1차** | DI water 세정 (HogoMax + debris 동시 제거) | 30초 |
| **가공 후 2차** (EMC 가공 시) | Chemical deflash + 고압 waterjet | STMicro QFN 실증 데이터. EMC 필러/탄화물은 DI water만으로 불충분 |
| **가공 후 보조** | CO2 스노우 클리닝 (잔류 나노파티클 제거) | 3-5nm까지 제거 가능하나 EMC debris 독립 검증 미확인 |

## 2. 재료별 핵심 위험과 대응

| 재료 | 최대 위험 | 부산물 특성 | 핵심 대응 |
| --- | --- | --- | --- |
| **Cu** | **전기적 단락** (전도성 나노파티클 재증착) | CuO/Cu2O 산화물, 나노\~서브마이크론 | N2 assist gas (산화 방지), 재증착 방지 필름, 즉시 흡입 |
| **EMC** | 필러 잔류물 + 유기 VOC | SiO2 파티클 + 에폭시 분해 가스 + 탄화물 | Chemical deflash 필수, 활성탄 필터 |
| **FR-4** | 탄화 + 유리섬유 분진 | 유리섬유(1725℃ 융점) + 에폭시 VOC | UV 레이저 권장, 수분 관리 |
| **Cu↔EMC 인터페이스** | **혼합 오염** (가장 위험) | 전도성 + 절연성 + 유기 혼합 debris | 인터페이스 통과 시 에너지 점진적 조정, 복합 세정 |

## 4. 상충점 해결 테이블

| 상충 항목 | R1 주장 | R3 주장 | 판단 |
| --- | --- | --- | --- |
| **최적 펄스폭** | ps = "품질+생산성 최적 균형" | fs = "EMC/Cu 정밀 가공 권장" | **양립 가능**. ps가 범용 최적, fs는 품질 최우선 시. 선택 기준: 택트타임 여유 + HAZ 허용 범위 |
| **fs 파티클 특성** | "가장 작은 나노입자" | "수밀도 높음, 응집체 5-10 um" | **동일 현상의 다른 프레이밍**. 1차 입자는 나노급이지만 응집체 형성. 실효 오염 크기는 응집체 기준으로 평가해야 함 |
| **EMC 후세정** | R2: CO2 스노우/DI water 충분 | R3: Chemical deflash + waterjet 필수 | **R3 우선 채택**. STMicro QFN 실증 데이터(Antilano, 2019)가 EMC 직접 사례. CO2 스노우는 보조 수단으로 위치 |

## 5. 역방향 의사결정 가이드

### 조건별 전략 선택

```
IF 패키지 구리층이 두꺼움 (>18 um)
  → Cu debris가 지배적 위험. N2 assist gas 필수, 재증착 방지 필름 필수
  → 파장: 532nm (Cu 흡수율 최적)

IF EMC 필러가 대형 (>20 um)
  → 직접 방출(ejection) 모드 우세, 더 높은 레이저 파워 필요
  → 물리적 흡입 시스템의 포집 효율이 핵심

IF 택트타임 여유 있음
  → 저에너지 멀티패스 + fs/short ps + UV(355nm) 조합
  → 최고 품질, debris 최소

IF 택트타임 제약 강함
  → 532nm + 500kHz + 8-10 패스 + N2 side-blow
  → 품질과 생산성 균형

IF 스트리트 폭 < 50 um
  → 보호 필름 두께(0.82-1.58 um)의 커프 영향 검토 필요
  → Coaxial 대신 side-blow 분사 필수 (좁은 커프에서 choking 심화)

IF 후세정 wet 공정 불허
  → CO2 스노우 클리닝 + 플라즈마 클리닝 (건식 조합)
  → 단, EMC 탄화물에 대한 효과 독립 검증 필요
```

### 결과 기반 조정

```
IF 가공 후 파티클 검사에서 Cu 재증착 발견
  → 1차: N2 유량 증가 (MSD 임계 이하에서)
  → 2차: 보호 필름 미사용이었으면 HogoMax 도입
  → 3차: 멀티패스 수 증가 (패스당 플루언스 감소)

IF 가공 후 EMC 탄화물 잔류 발견
  → 1차: 파장을 UV로 전환 (광화학 어블레이션 우세)
  → 2차: 플루언스 감소 + 패스 수 증가
  → 3차: Chemical deflash 후세정 강화

IF 커프 엣지 품질 불량 (거친 면, 크래킹)
  → 과도한 가스 압력 의심 → 압력 단계적 감소
  → 열축적 의심 → 반복률 감소 또는 패스 간 냉각 시간 추가
```

## 6. 예상 밖 핵심 발견

### 6.1 fs 레이저의 나노파티클 역설

"cold ablation"으로 알려진 fs 레이저가 **오히려 더 많은 수의 나노파티클을 생성**한다. HAZ와 용융 재증착은 극소화하지만, 미세 나노파티클의 수밀도가 높고 이들이 응집체(5-10 um)를 형성한다. "fs = 깨끗한 가공"이라는 통념은 **HAZ 관점에서만 사실**이며, 파티클 관리 관점에서는 반드시 그렇지 않다.

### 6.2 EMC 필러 크기가 어블레이션 전략을 좌우

EMC 내 실리카 필러의 크기에 따라 어블레이션 메커니즘 자체가 달라진다(증발 vs 직접 방출 vs 전이). 큰 필러는 더 높은 레이저 파워를 요구하며, 이는 부산물 특성도 변화시킨다. **EMC 재료 사양(필러 크기/함량)이 레이저 공정 파라미터의 독립 변수**임이 확인되었다.

### 6.3 가스 압력의 비단조적 효과

어시스트 가스 압력 증가가 항상 흄 배출을 개선하지 않는다. 특정 임계 이상에서 Mach Stem Disk, 경계층 분리, 외기 혼입이 발생하여 **오히려 품질이 악화**된다. 이는 "가스를 더 세게 불면 된다"는 직관에 반한다.

### 6.4 다층 인터페이스 = 최대 위험 구간

Cu→EMC 전환점에서 전도성/절연성/유기물 혼합 debris가 생성되며, 이는 **어떤 단일 세정법으로도 완전 제거가 어렵다**. 이 문제는 레이저 파라미터와 세정 기술 각각의 최적화만으로는 해결되지 않으며, 인터페이스 통과 시 에너지 프로파일의 점진적 조정이라는 공정 설계 수준의 접근이 필요하다.

## 7. Critic 지적 반영 및 한계

### 7.1 반영된 Critic 지적

| Critic 지적 | 본 보고서 반영 |
| --- | --- |
| 버스트 모드 "가장 효과적" 과도 확신 | 확신도 "낮음-중간"으로 교정, "ps 검증 필요" 명시 |
| ps vs fs 선택 기준 불명확 | 역방향 의사결정 가이드에서 조건별 분기 제시 |
| fs 파티클 "나노 vs 응집체" 이중성 | 예상 밖 발견 6.1에서 명확화 |
| EMC 후세정: R3 실증 &gt; R2 제품 자료 | Chemical deflash를 주(主), CO2 스노우를 보조로 위치 |
| 출처 없는 수치 (50-90% 감소, 10-50 sccm) | 근거 신뢰도 매트릭스에서 확신도 표기, 본문에서 미채택 |
| 2008년 데이터의 기술 세대 차이 | 멀티패스 권장에 "2008 ns 기반, ps 재검증 필요" 명시 |

### 7.2 본 리서치의 한계

1. **비용/생산성 정량화 부재**: 장비 가격, 공정 추가 비용, 택트타임 영향의 정량 비교가 없다. 의사결정자는 별도 벤치마크가 필요하다.
2. **경쟁 기술 대비 맥락 부재**: 블레이드/플라즈마/스텔스 다이싱 대비 레이저의 흄 문제 심각도를 비교하지 않았다. "왜 레이저인가"에 대한 답은 본 리서치 범위 밖이다.
3. **실패 모드 미분석**: 레이저 출력 드리프트, 보호 필름 코팅 불량, 필터 포화 등의 실패 시나리오 대응이 없다.
4. **대상 도메인 직접 데이터 부족**: 핵심 주장 다수가 인접 도메인(금속 판재, 배터리 전극, 분석화학)에서 도출되었으며, ps 레이저 + EMC/Cu 복합재료에서의 직접 실험 데이터는 제한적이다.

---

## 8. 후속 탐색 질문

1. **EMC 필러 크기/함량별 최적 레이저 파라미터 맵**: 현재 EMC는 단일 재료로 취급되었으나, 필러 사양(크기, 함량, 조성)에 따라 최적 조건이 달라진다. 실험적 파라미터 맵 구축이 필요하다.

2. **인터페이스 전환 감지 및 적응형 제어의 현재 기술 수준**: LIBS/광다이오드 기반 실시간 재료 감지 → 파라미터 자동 전환이 산업 수준에서 구현 가능한지, 현재 연구 격차는 어디인지.

3. **레이저 싱귤레이션 vs 경쟁 기술의 패키지 유형별 적합성 기준**: 어떤 패키지(두께, 재료, 스트리트 폭)에서 레이저가 최적이고, 어떤 조건에서 블레이드/플라즈마/스텔스 다이싱이 더 나은지.
