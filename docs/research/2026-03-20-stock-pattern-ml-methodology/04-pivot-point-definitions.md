# 04. 피봇 포인트 정량적 정의 — ML 탐지용 규칙 기반 패턴 사전

> **목적**: 스윙 트레이딩의 6대 돌파 패턴에 대해 ML/규칙 기반 시스템이 탐지·라벨링할 수 있는 정량적 파라미터를 정의한다.
>
> **대상 트레이더/저자**: William O'Neil, Mark Minervini, David Ryan, Kristjan Kullamägi(쿨라메기), Dan Zanger, TraderLion
>
> **출처 투명성**: 각 수치의 원출처를 명시하고, "이 수치가 틀릴 수 있는 조건"을 함께 기술한다.

---

## 0. 공통 전제 조건 — Minervini Trend Template (Stage 2 필터)

모든 패턴의 전제조건으로, Minervini의 8가지 Trend Template을 만족해야 한다. 이는 Stage 2 상승 추세에 있는 종목만을 대상으로 하기 위함이다.

| # | 조건 | 수치 기준 |
|---|------|-----------|
| 1 | 현재가 > 150일 SMA **그리고** > 200일 SMA | — |
| 2 | 150일 SMA > 200일 SMA | — |
| 3 | 200일 SMA가 최소 1개월(이상적으로 4~5개월) 상승 중 | 기울기 > 0 |
| 4 | 50일 SMA > 150일 SMA **그리고** > 200일 SMA | — |
| 5 | 현재가 > 50일 SMA | — |
| 6 | 현재가 >= 52주 최저가 대비 +30% 이상 | `(price - 52wLow) / 52wLow >= 0.30` |
| 7 | 현재가 <= 52주 최고가 대비 -25% 이내 | `(52wHigh - price) / 52wHigh <= 0.25` |
| 8 | RS Rating >= 70 (이상적 80~90+) | IBD RS 또는 자체 상대강도 계산 |

**출처**: Minervini, *Trade Like a Stock Market Wizard* (2011); Deepvue, Scribd 문서화; Finer Market Points 검증.

**이 수치가 틀릴 수 있는 조건**: 급격한 섹터 로테이션 시 RS Rating이 후행할 수 있다. IPO 후 6개월 미만 종목은 200일 SMA가 존재하지 않아 적용 불가하며, 별도의 IPO Base 기준이 필요하다.

## 1. VCP (Volatility Contraction Pattern)

### 1.1 정성적 정의

Mark Minervini가 체계화한 패턴으로, 기관 투자자의 매집이 완료되어 가는 과정에서 가격 변동폭이 점진적으로 축소되는 구조이다. 매도 압력(supply)이 소진되고 매수 수요(demand)가 유지되면서, 마치 스프링이 압축되듯 에너지가 축적된 후 돌파하는 형태를 보인다. David Ryan은 이를 "텐트를 치는 것"에 비유했다 — 각 기둥(수축)이 점점 짧아지면서 최종적으로 단단한 구조를 형성한다.

### 1.2 정량적 파라미터

#### 가격 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 수축 횟수 | 2~6회 (이상적 >= 3회) | Minervini; Finer Market Points (80% 성공 VCP에서 3회 이상) |
| 수축 비율 감소 | 각 수축이 이전 대비 약 20~30% 작아야 함 | Minervini |
| 수축 예시 | 1차 ~15% -> 2차 ~10% -> 3차 ~5% | Minervini 교과서적 예시 |
| 최대 첫 수축 깊이 | 일반 시장: <= 20~25%, 약세장: <= 35% | Minervini, TraderLion |
| 베이스 기간 | 최소 2~3주, 일반적 4~12주 | Minervini; Finer Market Points |
| Higher Low 형성 | 각 수축의 저점이 이전보다 높아야 함 | Minervini, TraderLion |

**이 수치가 틀릴 수 있는 조건**: 소형주/저유동성 종목에서는 변동성이 커서 수축 비율이 더 클 수 있다. 또한 곰 시장에서는 첫 수축이 35%까지 허용되나 성공률이 현저히 낮아진다.

#### 거래량 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 수축 중 거래량 | 각 수축마다 점진적 감소, 최종 수축에서 50일 평균의 40~60% 수준 | Minervini; Finer Market Points |
| 최종 수축 "드라이업" | 3~5일 연속 극도로 낮은 거래량 (거래 정지 의심 수준) | Minervini |
| 돌파 시 거래량 | 50일 평균 대비 >= 140~150% | Minervini; Finer Market Points |
| 반등 시 거래량 | 수축 중 반등에서는 거래량 증가 (기관 매집 신호) | Minervini |

#### 추세 조건

| 파라미터 | 기준 |
|----------|------|
| Stage 2 확인 | Trend Template 8개 조건 충족 (섹션 0 참조) |
| 사전 상승 | 베이스 형성 전 의미 있는 상승 존재 |
| 시장 환경 | 주요 지수가 200일 SMA 위, 상승 추세 |
| RS Rating | >= 70 (이상적 90+) |

### 1.3 피봇 포인트 정의

```
피봇 가격 = 최종(가장 타이트한) 수축 구간의 최고가
진입 신호 = 가격이 피봇 가격 + 1~2% 를 상향 돌파
              AND 돌파일 거래량 >= 50일 평균 x 1.4
손절선 = 최종 수축 구간의 최저가 바로 아래 (통상 피봇 대비 -3~7%)
```

**ML 구현 노트**: 피봇 가격은 마지막 수축의 `High`로 정의. 수축 구간은 가격 범위가 이전 수축보다 좁아지는 구간을 자동 감지하여 결정.

### 1.4 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공 (시원한 돌파)** | 피봇 돌파 후 10 거래일 내 +30% 이상 달성 |
| **보통 성공** | 피봇 돌파 후 20 거래일 내 +20% 이상, 손절선 미도달 |
| **실패 (Fake Breakout)** | 손절선(-7%) 먼저 도달, 또는 돌파 후 3일 내 피봇 아래 복귀 |

## 2. CWH (Cup-with-Handle)

### 2.1 정성적 정의

William O'Neil이 정의하고 Dan Zanger가 실전에서 가장 선호한 패턴이다. 강한 상승 후 U자형 조정(컵)을 거치고, 최종 소규모 눌림(핸들)에서 돌파하는 구조이다. V자형이 아닌 둥근 U자형 바닥이 핵심이며, 이는 점진적 매집과 약한 손의 퇴출을 의미한다. Zanger는 "시장 조정이 끝나가는 시점에서 컵앤핸들은 폭발적으로 움직이며, 가장 강력하고 신뢰할 수 있는 패턴"이라고 평가했다.

### 2.2 정량적 파라미터

#### 컵(Cup) 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 사전 상승 | 패턴 형성 전 최소 +30% 상승 | O'Neil, StockCharts |
| 컵 깊이 | 이전 고점 대비 12~33% 조정 (일반 시장) | O'Neil |
| 컵 깊이 (변동성 시장) | 최대 1/2~2/3 조정 허용 (극단적 경우) | O'Neil |
| 컵 기간 | 7~65주 (일반적 8~12주) | O'Neil; Zanger (8~12주) |
| 컵 형태 | U자형 (V자형은 이상적이지 않으나 핸들 존재 시 허용) | O'Neil; TraderLion |
| 양 림(Rim) 높이 | 좌측 고점 근사= 우측 고점 (대략 동일 수준) | O'Neil |

**Zanger 추가 기준**: 컵 조정 깊이 20~35%, 기간 8~12주. O'Neil보다 약간 좁은 범위를 선호.

#### 핸들(Handle) 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 핸들 깊이 | 컵 상승분의 1/3 이하 (이상적 10~15%) | O'Neil, StockCharts |
| 핸들 기간 | 1~4주 | O'Neil, StockCharts |
| 핸들 형태 | 하향 쐐기형(Descending Wedge) 선호 | O'Neil |
| 핸들 위치 | 베이스 상단 1/2 이상에 위치해야 함 | IBD |
| 핸들 거래량 | 형성 중 거래량 감소 (드라이업) | O'Neil, Zanger |

#### 거래량 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 컵 바닥 | 거래량 감소 (매도 소진) | O'Neil |
| 핸들 | 거래량 지속 감소 | O'Neil |
| 돌파 시 | 50일 평균 대비 +40~50% 이상 | IBD, O'Neil |

### 2.3 피봇 포인트 정의

```
피봇 가격 = 핸들 상단 추세선의 최고가 + $0.10 (또는 +0.1~0.5%)
진입 신호 = 가격이 피봇 가격을 상향 돌파
              AND 돌파일 거래량 >= 50일 평균 x 1.4
손절선 = 핸들 최저가 아래 (통상 피봇 대비 -5~8%)
```

**ML 구현 노트**: 핸들의 최고가는 핸들 구간 내 `High`의 최대값. 핸들 구간은 우측 림 도달 후 가격이 소폭 하락하는 1~4주 기간으로 정의.

### 2.4 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공** | 피봇 돌파 후 10일 내 +30% 이상, 또는 컵 깊이만큼의 추가 상승 달성 |
| **실패** | 손절선(-7~8%) 먼저 도달, 또는 돌파 후 핸들 아래로 복귀 |

## 3. HTF (High Tight Flag)

### 3.1 정성적 정의

O'Neil이 "가장 드물지만 가장 강력한 패턴"으로 정의한 구조이다. 극도로 강한 선행 상승(폴) 후 타이트한 횡보 조정(플래그)을 거쳐 재차 돌파하는 형태이다. 쿨라메기는 이 패턴을 자신의 3대 셋업 중 "Breakout" 카테고리의 핵심으로 사용하며, "20~50% 이상 움직인 후 눌리고 횡보한 뒤 다시 움직이는 것이 대부분의 리딩 종목 행동 양식"이라고 설명했다.

### 3.2 정량적 파라미터

#### 폴(Pole) 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 선행 상승률 (O'Neil 정의) | >= 100% (2배) in <= 8주 | O'Neil; Zacks; Deepvue |
| 선행 상승률 (쿨라메기 완화) | >= 30~100% in <= 12주 | 쿨라메기 (실전 범위) |
| 상승의 질 | 강한 거래량 동반, 갭업 포함 가능 | O'Neil, 쿨라메기 |

**수치 투명성**: O'Neil의 원래 정의(100%/8주)는 매우 엄격하여 출현 빈도가 극히 낮다. 쿨라메기는 30~50%+ 상승도 HTF/Breakout으로 취급하며, 이 완화된 기준이 실전에서 더 빈번하게 사용된다. TraderLion 역시 유사한 완화 기준을 채택한다.

#### 플래그(Flag) 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 조정 깊이 | <= 25% (이상적 10~20%) | O'Neil; Deepvue |
| 조정 기간 | 3~5주 (O'Neil), 2~8주 (쿨라메기) | O'Neil; 쿨라메기 |
| 가격 범위 | 타이트 -- 좁은 변동폭 (예: ~9% 이내) | Deepvue |
| 이동평균 지지 | 상승하는 10/20일 EMA 위에서 눌림 | 쿨라메기 |
| 50일 SMA | 멀지 않은 위치 (지지 가능 범위) | 쿨라메기 |

#### 거래량 조건

| 파라미터 | 기준 |
|----------|------|
| 플래그 중 | 거래량 지속 감소 (매도 압력 소진) |
| 상승일 거래량 | 하락일보다 높아야 함 |
| 돌파 시 | 평균 대비 강한 거래량 서지 |

### 3.3 피봇 포인트 정의

```
피봇 가격 = 플래그 구간의 최고가 (또는 폴 최고가, 둘 중 높은 값)
진입 신호 = 가격이 피봇 가격을 상향 돌파 + 거래량 확인
손절선 = 돌파일 장중 최저가 (쿨라메기), 또는 플래그 하단
```

**쿨라메기의 진입 스타일**: 풀 포지션을 돌파 시점에 한 번에 매수. 스케일링 인이 아닌 올인 방식. 손절은 돌파일 장중 저가 또는 전일 저가.

### 3.4 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공** | 피봇 돌파 후 10일 내 +30% 이상 |
| **실패** | 돌파 후 빠르게 실패 (강한 매도 거래량으로 피봇 아래 복귀), 또는 손절선 도달 |

## 4. W-Pattern (Double Bottom)

### 4.1 정성적 정의

O'Neil이 정의한 이중 바닥 패턴으로, 가격이 W자 형태로 두 번의 저점을 형성한 후 돌파하는 구조이다. 핵심 특성은 **두 번째 저점이 첫 번째 저점보다 약간 낮은 "언더컷"** 을 보여야 한다는 것이다. 이 언더컷은 약한 손의 최종 퇴출(shakeout)을 유도하며, 이후 반등은 진정한 수요 확인이 된다. IBD는 "컵앤핸들, 더블바텀, 플랫베이스가 가장 흔한 3대 패턴"이라고 강조한다.

### 4.2 정량적 파라미터

#### 베이스 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 사전 상승 | >= 30% | O'Neil, IBD |
| 전체 깊이 | 좌측 고점에서 두 번째 저점까지 <= 40% | IBD |
| 베이스 기간 | 최소 7주 (일반적 7~12주+) | IBD |
| 두 저점 관계 | 2차 저점 <= 1차 저점 (언더컷) | O'Neil -- 핵심 조건 |
| 언더컷 폭 | 통상 1차 저점 대비 1~5% 이내 | IBD 실전 관찰 |

#### 중간 고점(Middle Peak) 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 위치 | 베이스 상단 1/2 이상에 위치 | IBD |
| 좌측 고점 대비 | 좌측 고점(림)보다 약간 낮아야 함 | IBD |

#### 거래량 조건

| 파라미터 | 기준 |
|----------|------|
| 1차 저점 -> 중간 고점 | 반등 시 거래량 증가 |
| 중간 고점 -> 2차 저점 | 하락 시 거래량 감소 |
| 돌파 시 | 50일 평균 대비 +40~50% 이상 |

### 4.3 피봇 포인트 정의

```
피봇 가격 = 중간 고점(W의 중앙 봉우리) 최고가 + $0.10
진입 신호 = 가격이 피봇 가격을 상향 돌파
              AND 돌파일 거래량 >= 50일 평균 x 1.4
손절선 = 2차 저점 아래 (통상 피봇 대비 -7~10%)
```

**ML 구현 노트**:
- W 패턴 탐지 = 2개의 로컬 최저점 사이에 로컬 최고점 존재
- 2차 저점 < 1차 저점 (언더컷 확인)
- 피봇 = 중간 최고점의 High 값

### 4.4 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공** | 피봇 돌파 후 10일 내 +30% 이상 |
| **실패** | 2차 저점 아래로 재하락, 또는 돌파 후 중간 고점 아래 복귀 |

## 5. Low Cheat (조기 진입)

### 5.1 정성적 정의

Low Cheat은 독립된 패턴이 아니라, 위의 모든 패턴(VCP, CWH, HTF 등)에 적용되는 **진입 전략**이다. 정식 피봇 돌파 이전에, 베이스 하단 부근의 이동평균 지지에서 조기 매수하는 방식이다. Minervini는 "피봇보다 낮은 곳에서 속이는 것(cheat)"이라 표현했고, 쿨라메기는 10/20일 EMA 바운스를 Low Cheat의 핵심 기회로 본다.

**장점**: 피봇 대비 낮은 가격에 진입하여 리스크/리워드 비율 개선. 피봇까지의 거리가 추가 수익이 됨.

**단점**: 베이스가 완성되지 않아 실패할 위험이 높음. 더 높은 판별 능력 필요.

### 5.2 정량적 파라미터

#### 진입 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 패턴 맥락 | VCP/CWH/HTF 등의 베이스가 진행 중 (최소 수축 1회 이상 완료) | Minervini, 쿨라메기 |
| 이동평균 지지 | 가격이 상승하는 10일 EMA 또는 20일 EMA에서 지지 확인 | 쿨라메기 |
| Trend Template | Stage 2 필터 8개 조건 충족 | Minervini |
| Higher Low | 이전 수축 저점보다 높은 저점 형성 중 | Minervini |
| 거래량 | 눌림 중 거래량 수축 (드라이업) | Minervini, 쿨라메기 |
| 가격 범위 | 타이트한 일간 범위 (좁은 바) | Minervini |

#### 진입 시점 정의

```
Low Cheat 진입 = 아래 조건 모두 충족 시:
  1. Stage 2 Trend Template 통과
  2. 상위 패턴(VCP/CWH/HTF) 베이스 진행 중
  3. 가격이 상승하는 10일 또는 20일 EMA에 접근/터치
  4. 해당 접근에서 거래량 수축 (드라이업)
  5. Higher Low 형성 (이전 수축 저점보다 높음)
  6. 가격이 EMA에서 반등하며 장중 고가 돌파 또는 VWAP 위 복귀 시 매수

손절선 = 돌파일 장중 최저가, 또는 EMA 아래 (통상 피봇 대비 -3~5%)
```

**ML 구현 노트**: Low Cheat은 패턴 인식보다 **조건부 진입 타이밍**에 해당한다. 베이스 탐지 모델이 "진행 중" 상태를 출력한 후, 추가적인 MA 바운스 탐지 모듈이 진입 신호를 생성하는 2단계 구조가 적절하다.

### 5.3 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공** | 진입 후 정식 피봇까지 도달 + 피봇 돌파 성공 (전체로서 성공) |
| **부분 성공** | 정식 피봇 부근에서 매도하여 +5~10% 확보, 피봇 돌파 실패 |
| **실패** | EMA 지지 실패 -> 손절선(-3~5%) 도달 |

## 6. Episodic Pivot (EP)

### 6.1 정성적 정의

쿨라메기가 정의한 촉매(catalyst) 기반 돌파 패턴이다. 실적 발표, FDA 승인, 계약 체결 등 의미 있는 뉴스에 의해 대량 거래량과 함께 큰 갭업이 발생하는 것이 핵심이다. 이전의 소외/무관심(neglect) 기간이 길수록 EP의 폭발력이 크다. 쿨라메기는 "EP는 종종 멀티-위크 트렌드의 시작이 된다"고 설명하며, 이것이 그의 3대 셋업 중 하나임을 밝혔다.

기존 패턴들과의 핵심 차이: EP는 **사전 베이스가 필수가 아니다**. 촉매에 의한 급격한 가격 변화 자체가 신호이며, 이후 형성되는 플래그/베이스가 후속 진입 기회가 된다.

### 6.2 정량적 파라미터

#### 갭업/촉매 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 갭업 크기 | 전일 종가 대비 >= 10~20% (이상적 > 20%) | 쿨라메기 |
| 거래량 | 평균 거래량의 수 배 이상 (극단적 서지) | 쿨라메기 |
| 촉매 존재 | 실적 서프라이즈, FDA, 계약 등 구체적 뉴스 | 쿨라메기 |
| 사전 무관심 | 촉매 전 소외 기간(neglect) 존재 시 이상적 | 쿨라메기 |

#### 사전 상승 조건 (이상적, 필수 아님)

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 12주 내 상승 | 30~100%+ (이상적) | 쿨라메기 |
| 상대강도 | 상위 종목 (1~3개월 기준 최상위 퍼포머) | 쿨라메기 |

#### 진입 조건

| 파라미터 | 기준 | 출처 |
|----------|------|------|
| 장중 진입 | 시가 후 30분 Range High 돌파 시 | 쿨라메기 |
| 장중 가격 유지 | 갭업 후 시가 아래로 크게 밀리지 않아야 함 | 쿨라메기 |
| 후속 진입 | EP 후 형성되는 2~8주 플래그 돌파 시 (HTF/Breakout과 동일) | 쿨라메기 |

### 6.3 피봇 포인트 정의

```
EP 당일 피봇:
  피봇 가격 = 시가 후 첫 30분 고가 (Opening Range High)
  진입 신호 = 가격이 Opening Range High를 상향 돌파
  손절선 = EP 당일 장중 최저가, 또는 전일 종가

EP 후속 (PEAD, 플래그 돌파):
  피봇 가격 = EP 이후 형성된 플래그/베이스의 최고가
  진입 신호 = 일반 HTF/VCP 돌파와 동일
  손절선 = 플래그/베이스 하단
```

**ML 구현 노트**: EP 탐지는 가격 패턴보다 **이벤트 탐지**에 가깝다. 필요 데이터: (1) 일간 갭업 비율, (2) 거래량 급증 배수, (3) 뉴스/실적 데이터(NLP 또는 이벤트 플래그). 이후 PEAD(Post-Earnings-Announcement Drift) 플래그 형성을 모니터링하여 후속 진입 기회 포착.

### 6.4 성공/실패 판정

| 결과 | 조건 |
|------|------|
| **성공** | EP 후 10일 내 추가 +30% 이상 (PEAD 포함) |
| **실패** | 갭업 당일 시가 아래로 종가 형성 (갭 필), 또는 손절선 도달 |

## 7. 패턴 간 구분 기준 -- 중복 해결 규칙

실전에서 VCP, CWH, W-Pattern은 시각적으로 유사하여 중복 분류가 발생한다. ML 시스템에서는 아래 규칙으로 우선 분류한다.

### 7.1 VCP vs CWH 구분

| 기준 | VCP | CWH |
|------|-----|-----|
| 수축 구조 | 2회 이상 점진적 수축 (변동폭 감소) | 1개의 U자형 컵 + 1개의 핸들 |
| 형성 속도 | 상대적으로 빠름 (3~8주) | 상대적으로 느림 (7~65주) |
| 바닥 형태 | 날카로운 V형 수축 반복 | 둥근 U자형 단일 바닥 |
| 수축 횟수 | >= 2 (명확한 점진적 감소) | 1 (컵) + 1 (핸들) |

**분류 규칙**:
- 점진적 수축이 3회 이상이면 -> **VCP 우선**
- U자형 단일 바닥 + 핸들 구조이면 -> **CWH 우선**
- 2회 수축 + U자형 바닥이면 -> **양쪽 모두 태깅** (멀티라벨)

### 7.2 CWH vs W-Pattern 구분

| 기준 | CWH | W-Pattern |
|------|-----|-----------|
| 저점 수 | 1개 (U자형) | 2개 (더블 바텀) |
| 2차 저점 | 없음 | 1차 저점보다 약간 낮음 (언더컷) |
| 피봇 기준점 | 핸들 고가 | 중간 고점 |
| 핸들 유무 | 필수 | 선택적 (중간 고점이 피봇) |

**분류 규칙**:
- 명확한 2개 저점 + 언더컷 -> **W-Pattern**
- 단일 둥근 바닥 + 핸들 -> **CWH**

### 7.3 HTF vs VCP 구분

| 기준 | HTF | VCP |
|------|-----|-----|
| 선행 상승 | 극단적 (>= 100% / 8주, 또는 >= 30~50%) | 보통 (Stage 2 상승 중) |
| 조정 깊이 | 얕음 (<= 25%) | 중간 (첫 수축 15~25%) |
| 조정 기간 | 짧음 (2~5주) | 중간 (3~12주) |

**분류 규칙**:
- 선행 8주 수익률 > 80% 이고 조정 <= 25% -> **HTF 우선**
- 그 외 점진적 수축 -> **VCP 우선**

### 7.4 EP의 독립성

EP는 **촉매 기반**이므로 다른 패턴과 구분이 명확하다:
- 전일 대비 갭업 >= 10% + 거래량 3배 이상 + 뉴스 촉매 존재 -> **EP**
- EP 이후 형성된 플래그는 HTF/VCP로 추가 라벨링 가능 (듀얼 라벨: EP -> HTF)

## 9. 반증 탐색 및 한계점

### 9.1 패턴 기반 트레이딩의 반증

1. **승률의 한계**: 쿨라메기는 자신의 승률이 25~35%에 불과함을 공개했다. 이는 패턴 인식 자체보다 **포지션 사이징과 손절 관리**가 수익의 핵심임을 시사한다. 반증 미발견은 아니며, 패턴의 가치는 높은 R-배수(5~20x)에 있다.

2. **시장 환경 의존성**: Finer Market Points의 ASX 연구에 따르면, 시장 조건만으로 성공 확률이 50% 이상 변동한다. 곰 시장에서는 모든 패턴의 성공률이 급락한다.

3. **파라미터의 후행성**: 본 보고서의 모든 수치(12~33%, 140~150% 등)는 과거 성공 사례에서 역산된 것이다. 시장 구조 변화(알고리즘 트레이딩 증가, 유동성 변화)에 따라 최적 파라미터가 이동할 수 있다.

### 9.2 수치의 불확실성

| 수치 | 출처 | 틀릴 수 있는 조건 |
|------|------|-------------------|
| 수축 비율 20~30% 감소 | Minervini (경험적) | 높은 변동성 섹터(바이오텍 등)에서는 수축이 불규칙 |
| 돌파 거래량 140~150% | Minervini, O'Neil | 저유동성 종목에서는 소수 매수자로도 달성 가능 -> 허위 신호 |
| 컵 깊이 12~33% | O'Neil | 약세장에서는 40% 이상도 성공 사례 존재 |
| EP 갭 >= 10~20% | 쿨라메기 | 대형주에서는 5~8% 갭도 유의미할 수 있음 |
| 손절 -7% | O'Neil 표준 | 변동성 높은 시장에서는 너무 타이트하여 조기 손절 빈발 |

### 9.3 관점 확장

1. **시장 미시구조와의 결합**: 주문 흐름(order flow) 데이터를 결합하면 패턴 탐지의 정확도를 높일 수 있다. 특히 돌파 시점의 매수/매도 불균형은 거래량 단독보다 강력한 신호일 수 있다.

2. **시간 프레임 의존성**: 본 보고서의 모든 파라미터는 일봉(daily) 기준이다. 주봉(weekly)에서는 같은 패턴이 더 넓은 파라미터 범위를 가지며, 분봉(intraday)에서는 노이즈가 증가한다. ML 모델은 멀티-타임프레임 피처를 결합해야 한다.

3. **생존자 편향**: 이 보고서에서 인용한 트레이더들은 모두 성공한 사례이다. 동일한 패턴을 사용했으나 실패한 트레이더들의 데이터는 공개되지 않는다.

### 9.4 문제 재정의

> 원래 질문: "고확률 피봇 포인트를 정량적으로 정의하라"
>
> 더 적절한 핵심 질문: **"피봇 포인트의 정량적 정의 + 시장 환경 필터 + 포지션 사이징 규칙을 결합한 시스템에서, 각 요소의 상대적 기여도는 얼마인가?"**
>
> 쿨라메기와 Minervini 모두 패턴 인식보다 리스크 관리(포지션 사이징, 손절, 시장 환경 판단)가 수익에 더 큰 영향을 미친다고 명시적으로 말한다. 따라서 ML 시스템은 "패턴 탐지 정확도" 최적화보다 **"기대값(승률 x 평균 수익 - 패률 x 평균 손실) 최적화"** 를 목표로 해야 한다.

## 10. 의사결정 가이드

### "이 정보로 어떤 의사결정이 가능한가"

1. **라벨링 파이프라인 구축**: 섹션 1~6의 정량적 조건으로 과거 차트 데이터에 자동 라벨링을 수행할 수 있다. 각 패턴별 조건을 규칙 기반 필터로 구현하여 학습 데이터셋 생성.

2. **피처 엔지니어링**: 섹션 8의 피처 목록으로 ML 모델의 입력 피처를 체계적으로 구성할 수 있다.

3. **패턴 우선순위화**: HTF와 EP는 출현 빈도가 낮지만 R-배수가 가장 높다. VCP와 CWH는 빈도가 높지만 R-배수는 중간. 시스템의 목적(빈도 vs 수익률)에 따라 가중치를 다르게 설정해야 한다.

4. **멀티라벨 허용**: 섹션 7의 구분 규칙에도 불구하고, 실전에서는 하나의 베이스가 VCP이면서 CWH이기도 한 경우가 빈번하다. ML 시스템은 멀티라벨 분류를 허용하는 것이 현실적이다.

5. **시장 환경 게이트**: 패턴 신호가 발생하더라도, 시장 환경(지수의 Stage, 시장 폭) 필터를 통과하지 못하면 진입하지 않는 게이트 메커니즘이 필수적이다.
