# 아노다이징 피막 품질 판정: 산업 인증 체계 및 종합 점수화 사례

## 1. 주요 인증 체계별 판정 항목 정리

### 1.1 MIL-PRF-8625F (미군 사양)

미군 아노다이징 사양 MIL-PRF-8625F(구 MIL-A-8625)는 알루미늄 양극산화 피막에 대한 6가지 타입(Type I~IC, II, IIB, III)과 2가지 클래스(Class 1 무착색, Class 2 착색)를 정의한다. ★★★ [아노다이징 직접]

**판정 구조: 순수 Go/No-Go**

각 시험 항목에 대해 합격/불합격 기준(critical limit)이 명시되어 있으며, 종합 등급이나 점수는 부여하지 않는다. 하나라도 불합격이면 로트(lot) 전체가 거부된다.

| 시험 항목 | 적용 타입 | 합격 기준 | 불합격 기준 |
|-----------|-----------|-----------|-------------|
| **피막 두께** (Coating Thickness) | 전체 (Type II: 0.2~1.0 mil, Type III: >2.0 mil) | 에디커런트/마이크로미터 측정, 최소값 충족 | 최소값 미달 또는 불균일 |
| **내식성** (Salt Spray, ASTM B117) | 전체 (Type II: 168~336시간) | 규정 시간 내 허용 범위 이내 피팅 | 0.031인치 초과 피팅 또는 모재 부식 |
| **피팅 검사** (Post-Seal) | 전체 (5개 이상 시편, 150 sq in) | ≤15개 격리 피트, 30 sq in 내 ≤5개 | >15개 또는 0.031인치 초과 피트 |
| **봉공 품질** (Dye Stain/Weight Gain) | 전체 | 최소 염료 흡착, 규정 중량 증가 | 과도한 다공성/염료 흡착 |
| **내마모성** (Abrasion Resistance) | Type III 필수 | Taber 마모시험, 합금별 규정 사이클 | 규정 전 모재 노출 |
| **마모 지수** (Wear Index) | Type III | ≤1.5 mg/1000 cycles (Cu >2%: ≤3.5) | 기준 초과 |
| **밀착성** (Adhesion) | 지정 시 | 박리/탈락 없음 | 코팅 리프트 또는 플레이크 |
| **내광성** (Light Fastness, Class 2) | Type II/III Class 2 | ΔE ≤3 | ΔE >3 |
| **외관** (Appearance) | 전체 | 연속적, 균일, 밀착, 분말/스크래치 없음 | 불규칙, 변색, 분말 영역 |

**프로세스 관리**: 월 1회(≤35일 간격) 공정 관리 시험, 로트별 품질 적합성 시험 수행.

> **실행 연결**: MIL-PRF-8625F는 "이 피막이 합격인가 불합격인가"만 판단한다. "얼마나 좋은가"의 연속적 판단에는 사용할 수 없다. 종합 점수화를 위해서는 개별 시험의 측정값(두께, 마모량, ΔE 등)을 별도로 수집하여 정규화해야 한다.

**출처**: MIL-PRF-8625F 원문(coastlinemetalfinishing.com), Valence Surface Technologies, AAC(anodizing.org)

### 1.2 QUALANOD (유럽 품질 마크)

QUALANOD는 황산 아노다이징에 대한 국제 품질 라벨로, ISO 7599 및 ISO 10074을 기반으로 한다. 건축용(architectural), 장식용(decorative), 하드(hard), 산업용(industrial) 4가지 아노다이징 타입을 커버한다. ★★★ [아노다이징 직접]

**판정 구조: Go/No-Go + 두께 등급(Thickness Class)**

QUALANOD는 피막 두께에 따른 등급 체계를 사용하지만, 종합 점수는 부여하지 않는다. 각 시험 항목은 독립적 합격/불합격이다.

| 시험 항목 | 기준 | 합격 조건 |
|-----------|------|-----------|
| **외관 및 색상** (EN 12373-1) | 가시적 결함 검사 | 규정 거리에서 결함 없음 |
| **피막 두께** | 두께 등급별 (5, 10, 15, 20, 25 μm) | 전체 평균 ≥ 등급값, 개별 ≥ 80% |
| **봉공 품질 — 비파괴** (Dye Spot / Admittance) | EN 12373-4 / EN 12373-5 | 전 시료 최소 요건 충족 |
| **봉공 품질 — 파괴** (Weight Loss) | EN 12373-7 | 규정 중량 손실 이내 |
| **내마모성** (Abrasion) | 하드 아노다이징 | 규정 마모량 이내 |
| **내식성** (AASS 1000h / NSS 336h) | ISO 9227 | 규정 평가 기준 충족 |
| **내광성** | 착색 피막 | 규정 색차 이내 |

**라이선스 구조**: 초기 2회 검사(granting) → 연 2회 정기 검사 → 불합격 시 라이선스 정지/취소. 검사관이 공장 방문하여 시료 채취 및 시험.

**두께 등급 체계**: 5/10/15/20/25 μm 등급이 존재하나, 이는 "최소 요구 두께"의 카테고리이지 품질 점수가 아니다.

> **실행 연결**: QUALANOD 역시 종합 점수를 부여하지 않으며, 개별 항목의 Go/No-Go 판정 결과의 AND 조건으로 라이선스 유지 여부를 결정한다. 다만, 두께 등급은 고객-아노다이저 간 합의 기반 사양이므로, 더 높은 등급을 충족할수록 내구성이 높다는 간접적 서열화는 가능하다.

**출처**: QUALANOD Specifications 2026 Edition(qualanod.net), IFO GmbH(ifo-gmbh.de)

### 1.3 AAMA 611 (미국 건축용)

FGIA(구 AAMA)의 AAMA 611은 건축용 알루미늄 양극산화 피막의 시험 절차 및 요구사항을 정의한다. ★★★ [아노다이징 직접]

**판정 구조: 등급(Class I / Class II) + Go/No-Go**

AAMA 611은 2개 등급을 정의하며, 각 등급 내에서는 Go/No-Go 판정을 사용한다. 종합 점수는 없다.

| 항목 | Class I (고성능) | Class II (상업용) |
|------|------------------|-------------------|
| **피막 두께** (최소) | ≥0.7 mil (17.8 μm) | ≥0.4 mil (10.2 μm) |
| **내식성** (Salt Spray, ASTM B-117) | 3,000시간 후 ≤15 피트 | 1,000시간 후 ≤15 피트 |
| **봉공** (ADT, ASTM B-680) | <2.60 mg/in² | <2.60 mg/in² |
| **염료 스테인** (ASTM B-136) | 청색 염료 없음 | 청색 염료 없음 |
| **피막 중량** (ASTM B-137) | >27.0 mg/in² | >15.5 mg/in² |
| **피막 밀도** (ASTM B-137) | >38 g/in³ | >38 g/in³ |
| **마모 저항** (Michael Clarke Test) | 분필형 픽업 없음 | 분필형 픽업 없음 |
| **색상/광택** | ΔE <5 (10년 Florida), 광택 ±15 단위 | ΔE <5 (10년 Florida), 광택 ±15 단위 |
| **외관** | 3m 거리, 90° 각도에서 결함 없음 | 3m 거리, 90° 각도에서 결함 없음 |
| **보증** | 5~10년 | N/A |

> **실행 연결**: AAMA 611의 Class I/II 구분은 사실상 "두께 + 내식성 시간" 기반의 2단계 등급화이다. 종합 지수는 아니지만, 용도에 따른 최소 성능 등급을 제공한다. Class I은 외부 노출, Class II는 내부/경외부에 적합.

**출처**: AAMA 611-26 (FGIA), Linetec(linetec.com), Can Art Quality Specifications

### 1.4 GSB International (독일)

GSB International은 주로 **분체도장(powder coating) 전처리 및 코팅 재료**의 품질 인증을 담당한다. 아노다이징 자체보다는 아노다이징을 전처리(pre-anodizing)로 사용하는 도장 시스템의 품질을 관리한다. ★★ [인접 도메인: 분체도장]

**판정 구조: 3등급 계층(Standard / Master / Premium) + Go/No-Go**

GSB는 코팅 재료와 코팅 업체 모두에 3단계 등급을 부여한다. 이는 아노다이징 인증 체계 중 **가장 명확한 계층적 등급화**를 보여주는 사례이다.

| 등급 | Florida 노출 기간 | 총 UV 방사량 | 잔류 광택 요구 |
|------|-------------------|-------------|---------------|
| **Standard** | 약 12개월 | 300 MJ/m² | ≥50% 원래 광택 유지 |
| **Master** | 약 36개월 | 840 MJ/m² | 70~80% 광택 유지 (실측) |
| **Premium** | 약 60개월 | 1,400 MJ/m² | 70~100% 광택 유지 (실측) |

추가 라벨: **Sea Proof**, **Sea Proof Plus** (해양 환경 내식성)

**코팅 업체 등급**: Standard Coater → Master Coater → Premium Coater로, 기록 관리 수준, 현장 감사 빈도, 사용 가능한 재료 등급이 단계적으로 증가.

> **실행 연결**: GSB의 3등급 체계는 아노다이징 Robustness Index 설계에 가장 가까운 참고 모델이다. 단, GSB는 분체도장에 특화되어 있으므로 아노다이징에 직접 적용하려면 시험 항목과 가중치를 재구성해야 한다. 핵심 차이는 GSB가 "내후성(weathering resistance)" 단일 축을 기준으로 등급화한다는 점이며, 아노다이징에서는 내식성, 내마모성, 봉공 품질 등 다축 평가가 필요하다.

> **이 수치가 틀릴 수 있는 조건**: Florida 노출 시험 결과는 실제 사용 환경(산업 지역, 해안 지역, 극한 기후)과 다를 수 있다. GSB는 이를 보완하기 위해 Hoek van Holland(네덜란드 해안)에서 추가 야외 시험을 실시한다.

**출처**: GSB AL 631-7 / ST 663-7 (2026년 1월판), GSB International Info Letters

### 1.5 QUALICOAT (유럽 분체도장 품질 마크) — 비교 참고

QUALICOAT는 알루미늄 분체/액체도장의 품질 라벨로, GSB와 유사한 등급 체계를 사용한다. ★★ [인접 도메인: 분체도장]

| 등급 | Florida 노출 | 광택 유지 요구 |
|------|-------------|---------------|
| **Class 1** | 1년 | ≥50% |
| **Class 1.5** | 1.5년 | 중간 |
| **Class 2** ("Super Durable") | 3년 | ≥75% |
| **Class 3** | 10년 | ≥90% |

> **관점 확장**: QUALICOAT와 GSB 모두 내후성(광택 유지율)이라는 **단일 성능 축**을 기준으로 등급화한다. 아노다이징에서는 이런 단일 축 등급화가 존재하지 않는데, 이는 아노다이징 피막이 분체도장보다 **다차원적 성능**(내식성, 내마모성, 전기 절연, 외관)을 동시에 요구하기 때문일 수 있다.

## 2. 종합 점수화/지수화 시도

### 2.1 학술 논문: 단일 지수 통합 사례

아노다이징 품질을 **단일 지수**로 통합한 학술 논문은 발견되지 않았다. ★ [아노다이징 직접]

검색 결과, 아노다이징 관련 연구는 개별 성능 지표(피막 두께, 경도, 마모율, 다공성)를 독립적으로 측정하고 보고하는 구조를 취한다. 대표적으로:

- **마모 지수(Wear Index)**: MIL-PRF-8625F에서 mg/1000 cycles로 정의. 단일 값이지만 마모 성능만 반영.
- **경도**: Vickers (HV) 또는 Rockwell (HRF)로 측정. 아노다이징 후 모재 대비 5~8배 증가.
- **다공성**: SEM 분석으로 compact vs porous ("beehive") 구조 분류. 정량적 지수화 미시도.

> **반증 탐색**: "anodizing quality index", "anodizing composite score", "anodizing single metric" 등 다양한 키워드로 검색하였으나, 복수 시험 항목을 가중합하여 단일 지수로 통합한 학술 논문은 발견되지 않았다. **반증 미발견.**

> **문제 재정의**: 아노다이징 분야에서 종합 지수가 존재하지 않는 것은 "아직 시도되지 않았기" 때문이 아니라, 산업계가 **Go/No-Go 판정**으로 충분히 기능해왔기 때문일 수 있다. 종합 지수의 필요성은 "합격/불합격 이상의 품질 차별화"가 비즈니스 가치를 가질 때 발생한다.

### 2.2 특허: "Anodizing Quality Index" 관련

"anodizing quality index" 또는 유사 특허는 발견되지 않았다. ★

Apple의 아노다이징 관련 특허(예: 매트 블랙 피니시, L* <10 달성)는 **특정 외관 목표값**을 정의하지만, 복수 품질 항목을 통합한 지수는 아니다.

### 2.3 인접 도메인: 단일 품질 지수 사례

#### 2.3.1 Demerit Scoring System (감점 체계) — [인접 도메인: 제조 품질관리 일반]

제조 품질관리에서 가장 보편적인 종합 점수화 방법이다. ★★

**결함 분류 및 가중치**:
- **Class A (Critical)**: 100점 — 안전 위험/완전 기능 상실
- **Class B (Major)**: 50점 — 주요 기능 저하
- **Class C (Minor)**: 10점 — 경미한 성능 이슈
- **Class D (Cosmetic)**: 1점 — 외관 결함

**품질 지수 계산**:
```
D = (100 × nA + 50 × nB + 10 × nC + 1 × nD) / n
I = D / D₀
```
- I = 1: 기준 충족, I < 1: 기준 초과 달성, I > 1: 기준 미달

> **출처 평가**: Demerit scoring의 가중치(100/50/10/1)는 경험적으로 설정된 것이며, 특정 산업이나 제품에 최적화된 값이 아니다. 아노다이징에 적용 시 결함 유형별 가중치를 도메인 전문가 합의로 재설정해야 한다.

> **이 수치가 틀릴 수 있는 조건**: 가중치 비율(100:50:10:1)은 MIL-STD-1235 등에서 유래한 전통적 비율이나, 아노다이징에서 "외관 결함(cosmetic)"의 중요도는 용도에 따라 크게 달라진다. 소비전자 하우징에서는 cosmetic이 major와 동등하거나 더 중요할 수 있다.

**출처**: Research Inventy Vol.6 Issue 9, "Assess quality level of the final product by using Demerit system"

#### 2.3.2 Tupy SQI (Supplier Quality Index) — [인접 도메인: 자동차 주조/제조]

브라질 주조 기업 Tupy의 공급자 품질 지수는 **가중합 + 감점 승수(demerit multiplier)** 구조를 사용한다. ★★★

**SQI 공식 (2020년 5월 이후)**:
```
SQI = [(50% × MQI) + (20% × QSI) + (30% × CPI)] × (CAI/100) × (PRI/100) × (PQI/100) × (IED/100) × (IR/100)
```

| 구성 요소 | 약어 | 비중/역할 | 내용 |
|-----------|------|----------|------|
| 자재 품질 지수 | MQI | 50% (가중합) | 입고검사/적용/고객 불량 로트 수 기반 감점 승수 곱 |
| 품질시스템 지수 | QSI | 20% (가중합) | IATF 16949 인증=100%, ISO 9001만=50% |
| 상업 자세 지수 | CPI | 30% (가중합) | 협상 유연성, 견적 응답시간, 문제해결 민첩성, 비용절감 제안, 절차 준수 각 20점 |
| 시정조치 지수 | CAI | 승수 | 불적합 발생 위치별 감점 (고객 발견 시 더 큰 감점) |
| 신속 응답 지수 | PRI | 승수 | 시정조치 지연일 기반 감점 |
| 공정 품질 지수 | PQI | 승수 | 최근 공정 감사 점수 |
| 서류 전달 지수 | IED | 승수 | 서류 지연일 기반 감점 (일당 2%, 최대 20%) |
| 재발 지수 | IR | 승수 | 동일 불량 모드 재발 횟수 기반 감점 |

**MQI 감점 테이블** (불량 검출 위치별 감점 강도 차등):

| 불량 로트 수 | d2 (입고검사) | d3 (적용) | d4 (고객) | d5 (필드) |
|-------------|-------------|----------|----------|----------|
| 0 | 1.00 | 1.00 | 1.00 | 1.00 |
| 1 | 0.80 | 0.60 | 0.40 | 0.20 |
| 2 | 0.64 | 0.36 | 0.16 | 0.04 |
| 3 | 0.51 | 0.22 | 0.06 | 0.01 |
| 5 | 0.33 | 0.08 | 0.01 | 0.00 |

**분류 체계**: SQI 점수에 따라 A/B/C/D 등급 부여 (목표 ≥82점)

> **실행 연결**: Tupy SQI는 아노다이징 Robustness Index에 가장 직접적인 구조적 참고가 될 수 있다. 핵심 설계 원칙: (1) 일부 항목은 가중합(additive), (2) 일부 항목은 승수(multiplicative)로 작동하여 치명적 실패 시 전체 점수를 급격히 하락시킨다. 이는 Gate(승수=0) + Score(가중합) 구조와 동치이다.

> **관점 확장**: Tupy SQI에서 MQI의 감점이 검출 위치(입고→적용→고객→필드)에 따라 기하급수적으로 증가하는 구조는, 아노다이징에서도 "공정 내 검출 vs 고객 불만"의 차등 반영에 적용할 수 있다.

**출처**: Tupy Supplier Quality Index 문서 (tupy.com.br), 원문 extract로 검증

#### 2.3.3 반도체 — 다변량 SPC 통합 지수 — [인접 도메인: 반도체]

반도체 제조에서는 수백 개의 공정 변수를 **PCA(주성분분석)** 기반으로 압축하여 **단일 통계 지수(T² 통계량, Q 통계량)**로 모니터링한다. ★★

- Hotelling's T² 통계량: 정상 범위 내 변동의 종합 지수
- Q 잔차: 모델이 설명하지 못하는 이상 변동
- 두 지수를 결합하여 웨이퍼별 "건강도"를 단일 값으로 표현

> **출처 평가**: 반도체의 다변량 SPC는 "공정 파라미터"를 통합하는 것이지, "최종 제품 품질 시험 결과"를 통합하는 것이 아니다. 아노다이징 종합 지수와는 목적이 다르다. 다만, PCA 기반 차원 축소는 10개 이상의 시험 항목을 가진 아노다이징 품질 평가에서 가중치 결정의 데이터 기반 접근으로 활용 가능하다.

#### 2.3.4 MQM (Multidimensional Quality Metrics) — [인접 도메인: 번역/콘텐츠 품질]

번역 품질 평가의 MQM 시스템은 Gate + Score 하이브리드의 명확한 사례이다. ★★

- **오류 분류**: Critical / Major / Minor로 분류 (가중치: Critical=25, Major=5, Minor=1 등)
- **정규화 점수**: Normed Penalty = (가중합 × 1000) / 평가 단어 수
- **합격 임계값**: Quality Score = Maximum Score - Normed Penalty → Passing Threshold(예: 90점) 이상이면 PASS
- **캘리브레이션**: 원시 점수를 사용 편의를 위해 0~100 범위로 보정

> **실행 연결**: MQM의 구조는 아노다이징 종합 지수에 거의 그대로 적용 가능하다. (1) 결함 유형별 가중치, (2) 단위 면적당 정규화, (3) 합격 임계값 설정, (4) Critical 결함 시 자동 FAIL(Gate). 다만, MQM은 "결함 감점"만 반영하는 negative scoring이며, 아노다이징에서는 "봉공 품질", "내식성 시간" 등 positive metric도 통합해야 한다.

## 3. Gate vs Score 구조의 선행 사례

### 3.1 개념 정의

| 용어 | 정의 | 예시 |
|------|------|------|
| **Gate** (관문) | 불합격 즉시 전체 판정 실패. 보상 불가. | 안전 결함, 최소 두께 미달, 치명적 부식 |
| **Score** (점수) | 개별 항목의 측정값을 가중합하여 종합 점수 산출. 일부 항목의 낮은 점수를 다른 항목이 보상 가능. | 외관 균일성, 색차, 광택 |
| **Gate + Score** | 필수 관문 통과 후, 나머지 항목의 가중합으로 등급 결정. | HACCP + 품질 점수, MQM |

### 3.2 식품안전: HACCP CCP — [인접 도메인: 식품]

HACCP(Hazard Analysis and Critical Control Points)는 **Gate + (비공식) Score**의 원형이다. ★★★

- **CCP (Critical Control Point)**: 안전 위해 요인을 방지/제거/허용 수준으로 감소시킬 수 있는 공정 단계. **절대적 임계값(Critical Limit)**이 존재하며, 위반 시 즉시 시정조치 필요. → **Gate**
- **CQP (Critical Quality Point)**: 제품 품질에 영향을 주지만 안전과 무관한 단계. 점수화 또는 등급화 가능. → **Score**

**구조**:
```
Stage 1: CCP Gate → 모든 CCP의 Critical Limit 충족? (불합격 시 즉시 탈락)
Stage 2: CQP Score → 품질 관련 항목의 종합 평가 (등급 또는 점수)
```

> **실행 연결**: HACCP의 CCP/CQP 구분은 아노다이징에 직접 매핑 가능하다:
> - CCP(Gate): 최소 피막 두께, 봉공 무결성, 치명적 부식 → 하나라도 실패 시 즉시 불합격
> - CQP(Score): 색차(ΔE), 광택 균일성, 외관 미세 결함 수 → 가중합으로 품질 등급 결정

**출처**: FDA HACCP Principles & Application Guidelines, PQRI Training Guide

### 3.3 항공우주: AS9100 + Quality Gate — [인접 도메인: 항공우주]

AS9100 자체는 Quality Gate나 점수화를 표준으로 규정하지 않으나, 항공우주 산업에서는 **Phase Gate Review** (설계-제조-검증 각 단계의 관문 심사)가 광범위하게 사용된다. ★★

- 표준은 리스크 기반 사고를 요구하며, 조직이 자체적으로 Gate/Score 시스템을 설계
- 위험 항목(안전, 위조 부품 방지 등)은 필수 통과(mandatory gate)
- 성능 항목은 KPI 기반 모니터링 (score-like)

### 3.4 소프트웨어 코드 품질: Gate + Score 명시적 사례 — [인접 도메인: 소프트웨어]

최근 AI 코드 평가 프레임워크(arxiv 2603.15976)는 **명시적 3단계 Gate + Score 구조**를 사용한다: ★★

```
Stage 1: Gates (Hard Filters)
  - Compilation Gate: 컴파일 실패 → Score = 0, 평가 종료
  - Execution Gate: 실행 실패 → Score = 0
  - Memory Safety Gate: 메모리 누수 → Score = 0
  - API Usage Gate: 패턴 위반 → Score = 0

Stage 2: Metrics (정량 측정)
  - 실행 시간, 정확도, 수치 정밀도 등

Stage 3: Quality Assessment (가중합 점수)
  - 5개 카테고리(correctness, performance, code, appropriateness, library-specific)별 점수
  - 카테고리별 가중합 → 최종 점수
```

> **실행 연결**: 이 프레임워크는 아노다이징 Robustness Index의 **설계 템플릿**으로 사용할 수 있다:
> - Gate: 최소 두께, 봉공 무결성, 안전 관련 내식성 (불합격 시 Score = 0)
> - Metric: 각 시험의 원시 측정값 수집
> - Score: 측정값을 정규화하고 카테고리별 가중합 → 최종 Robustness Index

### 3.5 자동차: IATF 16949 + OEM Scorecard — [인접 도메인: 자동차]

IATF 16949 자체는 종합 점수를 정의하지 않으나, OEM들은 공급자 평가에 **자체 scorecard**를 사용한다: ★★

- Ford Q1 Scorecard
- GM Supplier Quality Excellence
- Tupy SQI (상기 2.3.2)

이들은 대부분 **가중합(품질, 납기, 비용) + 감점(치명적 불량 시)** 구조이다.

## 4. 종합 분석 및 의사결정 가이드

### 4.1 핵심 발견

1. **아노다이징 산업의 모든 주요 인증 체계는 Go/No-Go 구조**이다. 종합 점수나 지수를 부여하는 표준은 존재하지 않는다.

2. **등급화(tiering)는 존재하지만 점수화(scoring)와 다르다**:
   - AAMA 611: Class I / II (두께+내식성 기반)
   - QUALANOD: 두께 등급 5~25 μm
   - GSB/QUALICOAT: Standard / Master / Premium (내후성 기반)
   - 이들은 "어떤 수준을 충족하는가"의 이산적 등급이지, "얼마나 좋은가"의 연속적 점수가 아니다.

3. **종합 점수화는 인접 도메인에서 잘 확립되어 있다**:
   - Demerit scoring (제조 일반)
   - Tupy SQI (자동차 공급망)
   - MQM (번역 품질)
   - PCA 기반 통합 (반도체)

4. **Gate + Score 하이브리드 구조는 산업 전반에서 자연스럽게 사용**된다:
   - HACCP: CCP(Gate) + CQP(Score)
   - 소프트웨어: Compilation Gate + Quality Score
   - LLM 평가: Safety Gate + Weighted Criteria

### 4.2 근거 신뢰도 매트릭스

| 핵심 주장 | 출처 유형 | 도메인 일치도 | 확신도 | 검증 필요 |
|-----------|----------|-------------|--------|----------|
| 모든 아노다이징 인증은 Go/No-Go | 표준 원문 분석 | ★★★ 직접 | 높음 | 아니오 |
| 종합 지수 학술 논문 없음 | 검색 기반 | ★★ | 중간 | 예 (더 광범위한 검색) |
| Demerit scoring 적용 가능 | 일반 제조 | ★ 유추 | 중간 | 예 (도메인 적합성) |
| Tupy SQI 구조 참고 가능 | 자동차 공급망 | ★ 유추 | 높음 | 부분적 (가중치 재설정 필요) |
| Gate+Score 구조 범용성 | 다중 도메인 | ★★ | 높음 | 아니오 |

### 4.3 역방향 의사결정 가이드

- **아노다이징 Robustness Index를 설계한다면**:
  - Gate 항목: 최소 피막 두께, 봉공 합격, 부식 한계 → MIL-PRF-8625/QUALANOD 기준 직접 차용
  - Score 항목: 두께 여유율, 마모 지수, 색차, 광택 균일성, 봉공 품질 정량값 → 가중합
  - 가중치: 용도별 차등 (항공 → 내식성↑, 소비전자 → 외관↑, 건축 → 내후성↑)

- **결과가 "종합 지수가 불필요하다"이면** → 기존 Go/No-Go + 두께 등급만으로 충분하며, 추가적 품질 차별화가 비즈니스 가치를 창출하지 못하는 상황

- **결과가 "종합 지수가 필요하다"이면** → Tupy SQI 구조(가중합 + 감점 승수)를 템플릿으로, HACCP CCP/CQP 개념으로 Gate/Score를 분리하고, 도메인 전문가 합의로 가중치를 설정

### 4.4 예상 밖 핵심 발견

1. **GSB/QUALICOAT의 단일 축 등급화**: 분체도장은 "내후성"이라는 단일 축으로 등급화에 성공했다. 아노다이징은 다축 성능이 필요하므로 단일 축 등급화가 어렵지만, "주 용도별 핵심 축"을 정의하면 유사한 접근이 가능하다. 예: 건축용 → 내후성 축, 항공용 → 내식성 축, 소비전자 → 외관 축.

2. **Tupy SQI의 승수 구조**: 가중합만 사용하면 치명적 결함이 높은 점수에 묻힐 수 있다. Tupy의 승수 구조(Gate 실패 시 0으로 수렴)는 이 문제를 우아하게 해결한다. 아노다이징에서 "봉공 완전 실패"가 다른 모든 좋은 결과를 무효화하는 현실을 반영하기에 적합하다.

3. **OEM의 비공개 기준이 실질적 표준**: MIL-PRF-8625나 QUALANOD는 "최소" 기준이다. Apple, Samsung 등의 소비전자 OEM은 이보다 훨씬 엄격한 외관 기준을 적용하며, 이것이 실질적으로 아노다이징 산업의 품질 수준을 견인한다.

### 4.5 후속 탐색 질문

1. **아노다이징 시험 항목 간 상관관계**: 두께, 경도, 내마모성, 봉공 품질 간에 어느 정도 상관이 있는가? 높은 상관이 있다면 PCA로 차원 축소가 가능하고, 독립적이라면 모든 항목에 개별 가중치가 필요하다.

2. **국내 KS 표준(KS D 8301 등)의 판정 체계**: 한국 표준이 MIL-PRF-8625나 QUALANOD와 동일한 Go/No-Go인지, 아니면 등급화 요소가 있는지 확인 필요.

3. **실제 공정 데이터 기반 가중치 결정**: 도메인 전문가 합의가 아닌, 과거 불량 데이터와 고객 불만 데이터를 기반으로 가중치를 통계적으로 결정하는 접근이 가능한가?
