## 1. 비EIS 평가법 현황

### 1.1 내식성 시험

| 시험법 | 표준 규격 | 측정 대상 | 시험 시간 | 파괴/비파괴 | 비용 | 장단점 |
|--------|----------|----------|----------|-----------|------|--------|
| **염수분무(Salt Spray)** | ASTM B117, ISO 9227 NSS | 피막의 전반적 내식성 (핏팅, 백화) | 336h (아노다이징), 168h (크로메이트) | 파괴 | 중 (챔버 장비 필요) | ★ 산업 표준으로 비교 용이. ✗ 실제 환경과 상관성 낮음 — 습건 사이클 없이 연속 분무라 과대/과소 평가 가능 [아노다이징 직접] |
| **CASS 시험** | ASTM B368, ISO 9227 CASS | 가속 내식성 (장식용 코팅) | 16-48h (일반) | 파괴 | 중 | ★ ASTM B117 대비 3-6배 가속. ✗ CuCl₂ + CH₃COOH(pH 3.1-3.3)로 산성 환경 → 아노다이징보다 도금 평가에 최적화 [인접 도메인: 장식 도금] |
| **AASS 시험** | ISO 9227 AASS | 가속 내식성 | 48-96h | 파괴 | 중 | ★ NSS보다 가속. CH₃COOH로 pH 3.1-3.3. ✗ CASS보다 덜 공격적 |
| **인공땀 침지** | ISO 3160-2 (시계), EN 1811 (니켈 용출), DIN 53160 (색견뢰도) | 땀 접촉 부식/변색 | 24-72h (침지) | 파괴 | 저 | ★ 인체 접촉 소비전자에 직접 관련. ✗ 아노다이징 전용 규격 부재, 시계/귀금속 도메인에서 차용 [인접 도메인: 시계/귀금속] |

**인공땀 조성 (ISO 3160-2 기준)**: NaCl 20 g/L, NH₄Cl 17.5 g/L, 초산 5 g/L, DL-젖산 15 g/L, pH 4.7 (NaOH 조정), 온도 37±0.1°C ★★★

> **수치 투명성**: 염수분무 336h은 MIL-A-8625 Type II 봉공 아노다이징 기준. 두께, 봉공 방식, 합금에 따라 250-500h으로 달라질 수 있음 [아노다이징 직접]. 이 수치가 틀릴 수 있는 조건: 고실리콘 주조 합금(예: A380)이나 박막(< 10μm) 아노다이징.

| **HCl 침지(산 침지)** | — (사내 기준) | 봉공 상태 (버블링 관찰) | 5-10분 | 파괴 | 저 | ★ 극히 간단. ✗ 표준화 부족, 주관적 |

**어드미턴스와 EIS의 관계**: 어드미턴스(A_s, µS/cm²)는 EIS 등가회로의 다공층 커패시턴스와 직접 관련. A_s = 10 µS/cm² 이하이면 봉공 양호로 판정 ★★★. EIS R_pore와 역상관(R_pore 증가 → A_s 감소) [아노다이징 직접].

> **반증 탐색**: 어드미턴스 시험이 EIS를 대체할 수 있다는 주장에 대해 — 어드미턴스는 단일 주파수(1 kHz 영역) 측정으로, EIS의 다주파수 스펙트럼이 제공하는 배리어층 정보(R_barrier)와 주파수 의존 열화 패턴을 포착하지 못함. 반증 미발견: 어드미턴스만으로 충분하다는 학술적 주장은 확인되지 않음.

## 2. 다변량 결합 접근

### 2.1 복수 시험 결과 통합 방법론

#### 가중합(Weighted Sum) 접근 ★★

산업계에서 가장 직관적이고 널리 사용되는 방법. 각 시험 항목을 0-1로 정규화한 후 중요도 가중치를 곱하여 합산.

**제안 가중치 체계 (소비전자 외장용)**:

| 시험 항목 | 가중치 | 근거 |
|----------|--------|------|
| 봉공 품질 (어드미턴스/산용해) | 0.30 | 내구성의 근본 — 봉공 불량은 모든 후속 불량의 원인 |
| 내식성 (염수분무/인공땀) | 0.25 | 필드 불량의 주 원인 |
| 두께 | 0.20 | 내식성·내마모성의 기본 전제 |
| 외관 (색차/광택) | 0.15 | 소비전자 시각 품질 |
| 기계적 성질 (경도/마모) | 0.10 | 사용 중 스크래치 저항 |

> **수치 투명성**: 이 가중치는 소비전자 외장 아노다이징을 가정한 전문가 의견 기반 제안 ★. 항공/군사 용도에서는 내식성과 두께 가중치가 올라가고 외관은 낮아짐. 이 수치가 틀릴 수 있는 조건: 웨어러블 디바이스처럼 땀 접촉이 극심한 경우 내식성 가중치를 0.35 이상으로 올려야 할 수 있음.

#### PCA(주성분 분석) 기반 차원 축소 ★★

학술적 접근으로, 시험 항목 간 상관관계를 데이터 기반으로 파악하여 중복을 제거하고 소수의 주성분으로 품질을 표현.

- **장점**: 객관적 가중치 도출, 숨은 상관관계 발견
- **단점**: 충분한 샘플 데이터 필요 (최소 50-100건), 해석의 직관성 부족
- **아노다이징 적용 사례**: 직접 적용한 학술 문헌은 확인되지 않음. 유기코팅 품질 평가에서 PCA 사용 사례 존재 [인접 도메인: 유기코팅]

> **반증 탐색**: PCA가 가중합보다 우수하다는 주장에 대해 — PCA는 데이터 구조가 선형일 때 유효하며, 아노다이징 품질 지표 간 관계가 비선형(예: 두께-내식성의 포화 곡선)인 경우 과소적합 위험. 반증 미발견: 아노다이징 도메인에서 PCA vs 가중합을 직접 비교한 연구는 확인되지 않음.

#### 의사결정 트리/매트릭스 접근 ★

제조 현장에서 가장 실용적인 방법. 핵심 항목에 대한 pass/fail 게이트를 순차적으로 적용.

```
1단계 (즉시 불합격): 두께 < 규격 하한 → REJECT
2단계 (즉시 불합격): 봉공 시험 FAIL → REJECT
3단계 (조건부 합격): 개별 항목 점수 계산 → 종합 점수
4단계 (최종 판정): 종합 점수 ≥ 임계값 → ACCEPT
```

### 2.2 시험 항목 간 상관관계

#### 높은 상관 (중복 제거 가능성)

| 항목 A | 항목 B | 상관 근거 | 중복 수준 |
|--------|--------|----------|----------|
| 어드미턴스 (A_s) | EIS R_pore | 둘 다 다공층 전기적 특성 측정. A_s ↑ ↔ R_pore ↓ | 높음 ★★★ |
| 염색 반점(dye spot) | 산 용해(B680) | 둘 다 봉공도 평가. dye spot은 정성, B680은 정량 | 중-높음 ★★ |
| 피막 두께 | 염수분무 내구시간 | 두께 ↑ → 내식시간 ↑ (대략 선형, 봉공 양호 전제) | 중간 ★★ |

#### 낮은 상관 (독립적 정보)

| 항목 A | 항목 B | 이유 |
|--------|--------|------|
| EIS CDI% | 접착력 | CDI는 배리어층 결함, 접착력은 기재-피막 계면 — 다른 메커니즘 |
| 경도/마모 | 내식성 | 기계적 성질은 피막 밀도·구조, 내식성은 봉공 상태 + 결함에 주로 의존 |
| 색차(ΔE) | 봉공 품질 | 색 변화는 염료 안정성, 봉공은 구조적 밀봉 — 봉공 불량이면 색 변화도 동반하지만 역은 불성립 |

### 2.3 EIS 지표와 기존 시험법의 상관관계

#### |Z|₀.₁Hz vs 염수분무 시간

Buchheit et al. (Sandia National Labs, 1996)의 연구 ★★★:
- 33종 크로메이트 변환 피막 + 5종 알루미늄 합금에서 EIS와 168h 염수분무 비교
- 코팅 저항(R_c, ≈ |Z| at low frequency)과 염수분무 합격 확률 사이에 **로그-선형 관계** 확인
- R_c ≥ 2×10⁶ ~ 5×10⁶ Ω·cm² 이면 168h 염수분무 합격 확률 높음
- EIS가 염수분무보다 **극단값 영역에서 더 변별력 있음** (매우 좋거나 매우 나쁜 코팅 구분)

> **출처 평가**: [인접 도메인: 크로메이트 변환 피막] 이 연구는 아노다이징이 아닌 크로메이트 변환 코팅(conversion coating) 대상. 아노다이징 피막은 두께가 10-25µm로 변환 피막(0.1-1µm)의 수십 배이며, 등가회로 구조가 다름(2-layer vs 1-layer). R_c 임계값을 아노다이징에 직접 적용할 수 없으나, "EIS가 염수분무보다 변별력 있다"는 정성적 결론은 유효할 가능성 높음.

#### 어드미턴스 vs EIS 파라미터

Al 6061 아노다이징 연구 ★★★:
- 특이 어드미턴스(A_s)와 breakpoint frequency(f_b)가 365일 NaCl 침지 기간 동안 안정 → 양호 봉공 확인
- A_s = 10 µS/cm² 이하: 봉공 양호 판정 기준
- R_pore > 200 kΩ·cm²: 양호 아노다이징/봉공 기준
- A_s와 EIS 스펙트럼의 고주파 영역 응답은 동일 물리량(다공층 커패시턴스/저항) 측정

> **실행 연결**: 현장에서 어드미턴스 시험(2분)으로 봉공 품질 스크리닝 → 불합격 의심 시 EIS(30분)로 상세 진단. 이 2단계 접근이 비용 효율적.

## 3. 동남아 고온고습 + 인체 땀 환경

### 3.1 환경 조건 정의

| 파라미터 | 동남아 전형값 | 비고 |
|----------|-------------|------|
| **온도** | 28-35°C (연평균 27°C) | 극한: 40°C+ |
| **상대습도** | 70-90% RH (연평균 80%) | 우기 95%+ |
| **연간 강수량** | 1,500-3,000 mm | 습건 사이클 빈번 |
| **일사량** | 4.5-5.5 kWh/m²/day | UV 열화 가속 |
| **대기 오염** | SO₂, NOₓ (도시), 해안 NaCl | 복합 부식 |

**인체 땀 조성** ★★★:

| 성분 | 농도 | 역할 |
|------|------|------|
| 수분 | ~99% | 전해질 용매 |
| NaCl | 0.1-0.5% (1-5 g/L) | 핏팅 부식의 주 원인 (Cl⁻) |
| 젖산(Lactic acid) | 0.1-0.3% | pH 저하, Al 킬레이트 |
| 요소(Urea) | 0.1-0.2% | 가수분해 → NH₃ → pH 국소 상승 |
| NH₄⁺ | 미량 | 약한 부식성 |
| pH | 4.0-6.8 (평균 5.5) | 약산성 ~ 중성 |
| 온도 | 37°C (체온) | 반응속도 가속 |

> 인공땀 국제 표준(ISO 3160-2): NaCl 20 g/L, NH₄Cl 17.5 g/L, 초산 5 g/L, DL-젖산 15 g/L, pH 4.7 — 실제 땀보다 NaCl이 4-20배 높은 가속 조건 ★★★.

### 3.2 아노다이징 피막 열화 메커니즘

#### 3.2.1 고온고습 환경에서의 수화/가수분해

1. **수분 확산**: 습도 70%+ 환경에서 수분이 봉공이 불완전한 다공층 공극으로 침투
2. **수화 반응**: Al₂O₃ + nH₂O → Al₂O₃·nH₂O (n=1: 보에마이트 boehmite, n=3: 바이어라이트 bayerite)
3. **체적 팽창**: 보에마이트 형성 시 ~33% 팽창, 삼수산물 형성 시 ~100% 팽창
4. **구조 변형**: 비정질 알루미나 → 미세결정 보에마이트. AlO₄(사면체) → AlO₆(팔면체) 배위 전환
5. **황산염 방출**: 봉공 과정에서 잔류한 SO₄²⁻가 가수분해로 용출, 추가 공극 생성

**열화 시퀀스**:
```
고습도 노출 → 봉공 미비 영역 수분 침투 → 수화물 팽창 → 미세 균열 →
Cl⁻/유기산 침투 경로 생성 → 국부 부식 → 외관 결함 (백화/핀홀)
```

> **수치 투명성**: 33% 팽창은 Al₂O₃ → AlOOH (보에마이트) 변환의 이론 밀도 차이 기반 계산값 ★★. 실제 피막에서는 다공 구조가 팽창을 부분 흡수하므로 국부 응력으로 나타남. 이 수치가 틀릴 수 있는 조건: 봉공이 완벽한 경우 수분 침투 자체가 차단되어 수화 반응이 일어나지 않음.

#### 3.2.2 땀 접촉에 의한 봉공층 손상

1. **Cl⁻ 침투**: 땀의 NaCl이 봉공층의 수화물을 국부적으로 용해 → Al-O 결합 파괴 → 가용성 AlCl₃ 형성
2. **젖산의 킬레이트 작용**: 젖산이 Al³⁺와 안정한 착물 형성 → 산화피막의 균일 용해(uniform dissolution) 촉진
3. **pH 효과**: 땀의 약산성(pH 4-6)은 알루미나의 안정 영역(pH 4-9) 하한에 위치. pH < 4인 경우(운동 직후 땀) 봉공층 직접 용해 가능
4. **습도 상승 효과**: 75% RH 이상에서 땀 잔여물의 NaCl이 수분 흡수(조해) → 지속적 전해질 층 유지 → 부식 연속 진행 ★★★

> **핵심 발견**: 40% RH 미만에서는 땀 잔여물이 건조되어 부식이 정지됨. 동남아 환경(70-90% RH)에서는 땀 잔여물이 건조되지 않아 **연속 부식 조건**이 성립 — 이것이 동남아에서 아노다이징 필드 불량이 더 빈번한 핵심 이유. ★★

#### 3.2.3 염화물 + 유기산의 시너지 부식

- Cl⁻ 단독: 핏팅(국부) 부식
- 젖산 단독: 균일 용해 (느림)
- **Cl⁻ + 젖산 복합**: 젖산이 피막을 전반적으로 약화시키고, Cl⁻가 약화된 지점에서 핏팅을 개시 → **시너지 효과**로 단독 대비 부식 속도 2-5배 가속 ★ [전문가 의견/유추]

> **반증 탐색**: 시너지 가속 계수 "2-5배"의 정확한 출처는 아노다이징 피막 대상 연구에서 확인되지 않음. 유기코팅과 베어 알루미늄 부식 연구에서의 유추값. 반증: 잘 봉공된 두꺼운(20µm+) 아노다이징 피막에서는 시너지 효과가 미미할 수 있음.

#### 3.2.4 피막 박리(Delamination) 메커니즘

아노다이징 피막은 기재에서 직접 성장하므로 유기도막과 달리 **전체 박리는 드묾**. 그러나 다음 조건에서 국부 박리 발생:

1. **에지/코너**: 반경이 아노다이징 두께의 10배 미만인 날카로운 모서리에서 피막 균열·박리 (iPhone 17 Pro 사례: 카메라 플래토 에지에서 아노다이징 박리 보고) ★★★
2. **합금 내 금속간 화합물**: 2000계(Al-Cu), 7000계(Al-Zn-Mg) 합금의 CuAl₂, MgZn₂ 등 금속간 화합물 주위에서 불균일 산화 → 국부 결함 → 수분 침투 → 언더컷팅 박리
3. **열 사이클**: 알루미늄(CTE ~23 µm/m·°C)과 Al₂O₃(CTE ~8 µm/m·°C)의 열팽창 계수 차이 → 온도 변화 시 계면 전단 응력 → 반복 피로

> **관점 확장**: 소비전자에서 아노다이징 합금 선택(6063 vs 7000계)이 열대 환경 내구성에 큰 영향. 7000계는 강도 우수하나 금속간 화합물이 많아 핏팅 부식 감수성이 높음 — iPhone 17 Pro의 7000계 알루미늄 사용은 강도-내식성 트레이드오프.

#### 3.2.5 변색(Discoloration) 메커니즘

| 변색 유형 | 외관 | 메커니즘 | 주 원인 |
|----------|------|---------|---------|
| **백화(White spots)** | 흰색 점/반점 | 합금 내 Zn/Cu 산화물 개재물, 또는 봉공 부산물(Ni(OH)₂, Al(OH)₃) 석출 | 전처리 불량 + 습도 노출 |
| **지문 자국(Fingerprint corrosion)** | 지문 형상 부식 | 땀의 NaCl + 젖산이 봉공 불완전 영역에서 국부 부식 | 인체 접촉 + 고습도 |
| **얼룩(Staining)** | 불규칙 변색 | 세척수 오염물(Cl⁻, 조류), 봉공액 잔류물의 불균일 건조 | 제조 공정 결함 |
| **색 번짐(Color bleeding)** | 염색 경계 번짐 | 봉공 불량으로 염료가 이동·용출 | 봉공 부족 + 습도 |
| **황변(Yellowing)** | 투명 아노다이징 황변 | UV에 의한 봉공 수화물 변색, 또는 합금 내 Fe/Si의 산화 | UV + 고온 |

### 3.4 사전 예측 시험법 평가

#### 인공땀 침지 시험의 아노다이징 적용 가능성

| 표준 | 원래 도메인 | 아노다이징 적용 가능성 | 판단 근거 |
|------|-----------|---------------------|----------|
| ISO 3160-2 | 시계 케이스 (금속) | **높음** | 조성이 실제 땀과 유사, 아노다이징 피막도 금속 산화물 → 직접 적용 가능 |
| EN 1811 | 피부 접촉 니켈 용출 | **중간** | 니켈 특화이나 땀 조성 자체는 범용. 아노다이징 봉공에 니켈 봉공 사용 시 관련성 높음 |
| DIN 53160 | 색견뢰도 (장난감 등) | **높음** | 착색 아노다이징의 색 이행/견뢰도 평가에 직접 적용 가능 |

> **문제 재정의**: "아노다이징 전용 인공땀 시험 표준이 없다"는 것이 산업의 실제 갭. ISO 3160-2를 기반으로 아노다이징 특화 판정 기준(합격/불합격 임계값, 시험 시간, 평가 항목)을 수립하는 것이 더 적절한 핵심 질문.

#### 가속 습열(85°C/85% RH)과 열대 환경의 상관성

- **85/85 시험**: 반도체/전자부품 신뢰성 시험(JEDEC JESD22-A101)에서 유래. 1000h = 열대 옥외 10년 가속 비율로 사용되나, **아노다이징 피막에 대한 직접 상관 데이터는 확인되지 않음** ★
- **가속 계수의 불확실성**: 85°C는 아노다이징 피막의 수화 반응을 과도하게 가속(열수 봉공 조건 ~100°C에 근접)하여, 실제 열대 환경(30-35°C)에서는 일어나지 않는 열화 모드를 유발할 수 있음
- **권장**: 60°C/90% RH (IEC 60068-2-78 변형)가 아노다이징 피막에는 더 현실적인 가속 조건. 봉공층의 비정상적 수화를 방지하면서도 충분한 가속 효과

> **반증 탐색**: 85/85가 아노다이징에 부적합하다는 직접 연구는 확인되지 않음. 그러나 85°C에서의 알루미나 수화 속도가 30°C 대비 비선형적으로 가속되므로(Arrhenius 관계), 상관성 외삽에 주의 필요. 반증 미발견.

#### 필드 불량 예측력 비교 ★

| 시험법 | 필드 불량 예측력 | 근거 |
|--------|----------------|------|
| 어드미턴스 + 두께 | **높음** | 봉공 불량은 대부분 필드 불량의 선행 조건 |
| 인공땀 침지 (72h) | **높음** | 소비전자 사용 환경 직접 시뮬레이션 |
| 염수분무 (336h) | **중간** | 과대 가속, 실제 환경과 괴리 |
| EIS (|Z|₀.₁Hz + CDI%) | **높음** | 봉공 + 배리어층 정보 동시 제공 |
| 순환 습열 (IEC 60068-2-30) | **높음** | 열대 사이클 조건 재현 |

| **외관 검사** | 육안 + ΔE 측정 | 30초 | ~$0.2 | ΔE ≤ 1.0 (동일 로트 내), 결함 없음 |

### 4.2 판정 흐름도

```
[전수] 두께 OK? ──No──→ REJECT
         │Yes
[전수] 어드미턴스 ≤ 20 µS/cm²? ──No──→ HOLD (Tier 2 추가 시험)
         │Yes
[전수] 외관 OK? ──No──→ REJECT (외관 불량)
         │Yes
         ├── PASS (출하 가능)
         │
[샘플링] Tier 2 결과 → 전체 합격 → PASS
                      → 1개 이상 불합격 → HOLD + 추가 샘플링 또는 REJECT
```

### 4.3 파괴 vs 비파괴 샘플링 전략

| 검사 단계 | 방식 | 샘플링 | 근거 |
|----------|------|--------|------|
| Tier 1 | 비파괴 | 전수 (100%) | 저비용·신속, 즉시 판정 |
| Tier 2 | 파괴 | AQL 샘플링 (MIL-STD-1916 등), 일반 n=3-5/로트 | 비용 대비 통계적 신뢰 |
| Tier 3 | 파괴 | 초기 자격 인증 시 n=10+, 이후 연 1-2회 또는 공정 변경 시 | 공정 능력 확인 |

### 4.4 동남아 출하 제품의 추가 권장 시험

동남아 열대 환경 타겟 제품은 표준 시험 세트에 **아래를 추가 권장**:

1. **인공땀 침지 72h** (ISO 3160-2 용액, 37°C): 로트당 n=3. 변색·핏팅 판정
2. **순환 습열 10사이클** (IEC 60068-2-30, 25-55°C): 초기 자격 인증 시 필수
3. **어드미턴스 기준 강화**: A_s ≤ 10 µS/cm² (표준 대비 엄격화)

## 6. 관점 확장 및 숨은 변수

### 인접 질문
1. **합금 선택의 영향**: 6063 vs 6061 vs 7075 — 동남아 환경 내구성 순위는? 금속간 화합물 밀도가 핏팅 감수성을 결정하므로, 합금 선택이 시험 결과보다 중요할 수 있음
2. **봉공 방식의 영향**: 열수 봉공 vs 니켈 봉공 vs 냉간 봉공 — 열대 환경에서 어떤 봉공이 최적인가? 니켈 봉공은 어드미턴스 값이 낮으나 니켈 용출 규제(EU REACH) 이슈

### 숨은 변수
- **세척 잔류물**: 제조 후 세척 불충분 → 잔류 산/알칼리가 열대 환경에서 활성화 → 이것이 아노다이징 자체의 문제가 아닌 후처리 문제인 경우가 상당수
- **포장/운송 중 결로**: 동남아 수출 시 해상 컨테이너 내 온도 변화 → 결로 → 아노다이징 제품 백화. 시험 스펙이 아니라 물류 관리 문제

### 문제 재정의
> "비EIS 시험법과 EIS를 어떻게 결합할 것인가"보다 더 적절한 질문: **"제조-출하-사용 전 생애주기에서 어떤 시점에 어떤 시험을 배치하면 동남아 열대 환경 필드 불량을 최소화할 수 있는가"** — 시험법 결합이 목적이 아니라 필드 불량 예방이 목적이므로, 시험 타이밍과 판정 기준의 환경 특화가 핵심.
