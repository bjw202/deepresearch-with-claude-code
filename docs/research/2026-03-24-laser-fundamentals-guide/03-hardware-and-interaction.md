## 1. 레이저 시스템 전체 구성

### 1.1 블록 다이어그램 — "가공 라인의 신호 흐름"

절삭가공 기계에 빗대어 보면:

| 레이저 시스템 | 절삭가공 대응 개념 |
| --- | --- |
| 레이저 소스 (Laser Source) | 스핀들 모터 (주축) |
| 빔 전달계 (Beam Delivery) | 주축 → 공구홀더 전달 경로 |
| 가공 헤드 (Processing Head) | 공구홀더 + 인서트 |
| 모션 시스템 | CNC 축 (X/Y/Z) |
| 제어기 (Controller) | CNC 컨트롤러 (G코드 실행) |
| 어시스트 가스 | 절삭유·에어블로 |
| 집진기 (Fume Extractor) | 칩 컨베이어 + 쿨런트 회수 |
| 냉각기 (Chiller) | 주축 냉각·기계 냉각 |

```
[전원 공급 (Power Supply)]
        ↓
[레이저 소스 (Laser Source)] ←→ [냉각기 (Chiller)]
        ↓ 빔(광섬유 또는 공간)
[빔 전달 광학계 (Beam Delivery Optics)]
  · 콜리메이터 (Collimator)
  · 빔 익스팬더 (Beam Expander)
  · 편향 미러 (Steering Mirror)
        ↓
[가공 헤드 (Processing/Cutting Head)]
  · 집광 렌즈 (Focusing Lens)
  · 노즐 (Nozzle)
  · 크로스젯 (Cross-jet, 렌즈 보호)
        ↓ 빔 + 어시스트 가스
[피가공물 (Workpiece)]
        ↑
[모션 시스템 (Motion System)]
  · XY 갠트리 / 갈바노 스캐너 / 5축 로봇
        ↑
[제어기 (Controller)]
  · CNC / PLC / 레이저 전용 제어기
  · PSO, 변조 신호

[안전 시스템 (Safety/Interlock)] — 모든 구성 요소 감시
[배기/집진 시스템 (Fume Extraction)] — 가공부 주변
```

### 1.2 레이저 소스 종류별 시스템 구성 차이

| 소스 종류 | 빔 전달 방식 | 주요 하드웨어 특징 |
| --- | --- | --- |
| 파이버 레이저 (Fiber Laser) | 광섬유 케이블 | 소형, 수냉, 파이버 코어 직경이 빔 품질 결정 |
| CO₂ 레이저 | 미러 + 빔 파이프 | 대형, 가스 보충 필요, 정렬 민감 |
| Nd:YAG / Nd:YVO₄ | 광섬유 또는 자유공간 | 플래시램프/다이오드 펌핑 |
| 초단펄스 (fs/ps) | 광섬유 또는 자유공간 | 별도 발진기+증폭기(CPA) 구조 |
| 다이오드 레이저 (Diode) | 광섬유 또는 직접 | 전기→빔 효율 높으나 빔 품질 낮음 |

## 2. 전원 공급 장치

### 2.1 기본 역할

레이저 소스에 전기 에너지를 공급하는 장치. 소스 종류마다 요구 사항이 다르다.

**절삭가공 비유**: 인버터 스핀들 드라이브와 같다. 입력(AC 전원)을 레이저 소스에 맞는 전기 형태로 변환한다.

### 2.2 소스별 전원 특성

| 레이저 소스 | 전원 공급 형태 | 특징 |
| --- | --- | --- |
| 파이버 레이저 | 안정 DC + 펌프 다이오드 구동 | 전기→광 변환 효율 25\~45%; IPG 등은 통합형 파워서플라이 내장 |
| CO₂ 레이저 | 고전압(수 kV) RF 또는 DC 방전 | 방전으로 CO₂ 가스 여기; RF 방전 방식이 현재 주류 |
| Nd:YAG (플래시램프) | 고전압 커패시터 방전 | 플래시램프 수명(수백만\~수천만 발) 관리 필요 |
| Nd:YAG (다이오드 펌핑) | 정전류 다이오드 드라이버 | 펄스 파형·전류 정밀 제어 가능 |
| 초단펄스 | 낮은 평균 전력, 전용 드라이버 | 피크 파워가 수십 GW여도 평균 전력은 수십 W에 불과 |

> **실행 연결**: 전원 품질(라인 노이즈, 전압 변동)이 출력 안정성에 직결된다. 대용량 레이저 설치 시 전용 UPS 또는 전압 조정기(AVR) 검토가 필요하다.

## 3. 냉각 시스템 (Chiller)

### 3.1 왜 냉각이 필요한가

레이저 소스의 전기→광 변환 효율은 완벽하지 않다. 나머지 에너지는 열로 발산된다.

- **파이버 레이저**: 변환 효율 \~30\~45% → 나머지 55\~70%는 열
- **CO₂ 레이저**: 변환 효율 \~10\~20% → 나머지 80\~90%는 열
- 열이 누적되면 열렌즈(Thermal Lensing) 발생 → 빔 품질 저하, 초점 이탈

**절삭가공 비유**: 주축 냉각 없이는 베어링 소손, 열팽창으로 가공 정밀도 저하. 레이저도 동일한 원리.

### 3.2 수냉 방식 (Water Chiller)

산업용 레이저의 대부분은 수냉식 칠러를 사용한다.

**냉각수 품질 요구사항** (이 부분은 현장에서 가장 많이 실수하는 영역)

| 항목 | CO₂ 레이저 | 파이버 레이저 | 이유 |
| --- | --- | --- | --- |
| 물 종류 | 증류수 (Distilled Water) | 탈이온수 (Deionized Water, DI) | 미네랄이 스케일·부식 유발 |
| 전도도 (Conductivity) | &lt;10 µS/cm | &lt;1\~5 µS/cm (제조사별 상이) | 이온이 내부 부품 부식 |
| pH | 6.5\~7.5 | 6.5\~7.5 | 중성 유지로 부식 최소화 |
| 교체 주기 | 3개월마다 | 3개월마다 | 세균·녹 발생 전 선제 교체 |
| 배관 세정 | 6개월마다 Citranox 1\~2% | 6개월마다 | 스케일 제거 |

> **주의**: 수돗물 사용 금지. 파이버 레이저 내부에는 구리/알루미늄 열교환기가 있어 이온 오염 시 전기 단락·부식 발생 가능.

**냉각 온도 설정**:

- 통상 레이저 소스: 15\~25°C (제조사 지정값 사용)
- 너무 낮게 설정하면 결로(Condensation) → 광학계 오염
- 너무 높게 설정하면 레이저 소스 과열, 출력 저하

**냉각 용량 선정 기준**: 레이저의 실제 열 발산량(Waste Heat = 입력 전력 × (1 - 변환 효율))의 120\~150% 여유를 갖는 칠러 선택.

### 3.3 공냉 방식 (Air-Cooled)

소출력(100W 이하) 레이저나 일부 초단펄스 레이저에서 사용. 구조 단순, 유지 쉬우나 고출력 불가.

## 4. 제어기 (Controller)

### 4.1 레이저 제어 신호의 종류

레이저 소스는 단순히 "전원 ON/OFF"가 아니라, 정밀한 신호로 제어된다.

**아날로그 신호 (Analog Signal)**

- 0\~10V 전압으로 출력 파워를 비례 제어
- 예: 5V → 50% 파워, 10V → 100% 파워
- 응답 속도: 수십\~수백 µs 수준

**디지털 PWM (Pulse Width Modulation)**

- 일정 주파수에서 펄스 폭(Duty Cycle)으로 평균 파워 조절
- 예: 주파수 20kHz, Duty 50% → 평균 파워 50%

**게이팅 (Gating)**

- 레이저 빔의 ON/OFF를 고속으로 제어
- 모션 시스템과 동기화: 감속 구간에서 자동 OFF → 과열/과가공 방지
- 직선 구간 ON, 코너 감속 구간 OFF

**변조 (Modulation)**

- 펄스형 레이저에서 펄스 주파수(Repetition Rate)·펄스 폭(Pulse Width)·파워를 실시간 변경
- 재료 두께, 가공 형상에 따라 파라미터를 동적으로 바꿈

### 4.2 PSO — Position Synchronized Output (위치 동기 출력)

레이저 가공에서 가장 중요한 제어 개념 중 하나.

**정의**: 모션 시스템의 실시간 위치(Position) 정보를 기반으로 레이저 출력을 동기화하는 기능.

**왜 필요한가**:

- 일반 시간 기반 제어: 모션 속도가 변해도 레이저는 일정 시간 간격으로 ON → 코너에서 과가공, 직선에서 부족
- PSO 제어: 모션 위치 x mm마다 레이저 펄스 1회 → 속도와 무관하게 균일한 가공

**절삭가공 비유**: 이송속도가 변해도 mm당 칩 로드(chip load)를 일정하게 유지하는 것과 같다.

**PSO 구현 예**:

- Aerotech, Beckhoff, KEBA 등의 모션 컨트롤러에 내장
- 레이저 소스의 Trigger/Gate 입력 단자와 컨트롤러를 연결

### 4.3 인터페이스 종류

| 인터페이스 | 특징 | 사용 사례 |
| --- | --- | --- |
| RS-232 / RS-485 | 직렬 통신, 느리지만 단순 | 레이저 파라미터 설정, 상태 읽기 |
| EtherCAT | 고속 실시간 통신, 1µs 수준 | 산업용 모션 컨트롤러와 통합 |
| 아날로그 0\~10V | 단순하고 빠름 | 파워 실시간 제어 |
| TTL/Gate 신호 | 디지털 ON/OFF | 게이팅, PSO 트리거 |
| 이더넷 (TCP/IP) | 파라미터 업로드, 원격 모니터링 | 공장 MES 연동 |

## 5. 인터록 시스템 (Interlock)

### 5.1 개념

**인터록(Interlock)**: 특정 조건이 충족되지 않으면 레이저 발진을 자동으로 차단하는 안전 회로.

**절삭가공 비유**: 안전 도어가 열리면 스핀들이 멈추는 도어 인터록과 동일한 개념. 레이저는 에너지 밀도가 훨씬 높아 더 엄격하다.

### 5.2 인터록 체인 구성

```
안전 인터록 체인 (Safety Interlock Chain)
  ├── 도어 스위치 (Door Switch) — 인클로저 문 열림 감지
  ├── 비상 정지 (E-Stop) — 수동 즉각 차단
  ├── 칠러 이상 (Chiller Fault) — 냉각수 흐름·온도 이상
  ├── 가스 압력 이상 (Gas Pressure Fault) — 어시스트 가스 압력 부족
  ├── 광학 온도 이상 (Optics Temp) — 렌즈·미러 과열
  ├── 출력 파워 이상 (Power Monitor) — 설정값 대비 이탈
  └── 외부 인터록 (External Interlock) — 로봇, PLC, 안전 PLC 연동

하나라도 이상 → 레이저 즉시 차단 + 경고등 점등
```

### 5.3 실무 주의사항

- 인터록을 임의로 단락(Short-circuit)하면 안전사고 직결 → 절대 금지
- 인터록 바이패스가 필요한 정비 작업은 잠금(LOTO, Lock-Out Tag-Out) 절차 준수
- 신규 설치 시 인터록 체인 전체 동작 검증 필수

## 6. 가스 시스템

### 6.1 CO₂ 레이저 발진용 가스

CO₂ 레이저는 발진 매질이 가스(CO₂ + N₂ + He 혼합)이므로, 가스를 지속적으로 보충하거나 순환시킨다.

- **밀봉형 (Sealed-off)**: 수명 내 가스 보충 불필요, 수명 이후 교체
- **흐름형 (Gas-flow)**: 연속 가스 공급으로 출력 안정성 유지

### 6.2 어시스트 가스 (Assist Gas)

레이저 절단에서 가공 품질과 효율을 결정하는 핵심 변수.

**기본 역할**:

1. 용융물(Melt)을 키르프(Kerf, 절단 홈) 밖으로 불어냄
2. 집광 렌즈를 스패터와 흄으로부터 보호
3. 산화·비산화 반응 제어

| 가스 종류 | 적합 재료 | 메커니즘 | 특징 |
| --- | --- | --- | --- |
| **산소 (O₂)** | 연강(Mild Steel), 탄소강 | 발열 산화 반응 → 절단 에너지 보조 | 빠른 절단, 단면에 산화막, 두꺼운 판에 적합 |
| **질소 (N₂)** | SUS(스테인리스), 알루미늄, 얇은 판 | 비반응성, 용융물만 불어냄 | 산화 없는 깨끗한 단면, 고압(15\~30 bar) 필요 |
| **압축 공기** | 박판(≤6mm) | N₂+O₂ 혼합 → 부분 산화 | 저비용, 중간 품질 |
| **아르곤 (Ar)** | 구리, 황동 등 반응성 금속 | 완전 불활성, 반응 방지 | 고비용, 특수 용도 |

**가스 압력 범위** (출처: Linde Laser Cutting Process Guide):

- O₂ 절단: 3\~10 bar
- N₂ 절단: 15\~30 bar (판 두께 증가 시 상승)

> **수치 주의**: 위 범위는 일반적 가이드라인. 실제 최적 압력은 기계, 노즐 직경, 재료 두께 조합에 따라 다르다. 반증 미발견.

**가스 순도**: N₂ 절단 시 99.95% 이상 권장. 순도 저하 → 단면 황변, 드로스 부착.

## 7. 배기/집진 시스템 (Fume Extraction)

### 7.1 왜 중요한가

레이저 가공 시 발생하는 흄과 파티클은:

- **작업자 건강**: 금속 산화물 나노입자, 유기 VOC (휘발성 유기 화합물) → 호흡기 위험
- **광학계 오염**: 렌즈·미러에 미세 입자 퇴적 → 레이저 투과율 저하, 열 흡수 → 광학 손상
- **기계 오염**: 모션 시스템, 전자부품에 도전성 분진 퇴적 → 오작동

### 7.2 집진 시스템 구성 (필터 다단계)

```
흄 발생원 (가공부)
    ↓ 포집 (캡처 후드 또는 인클로저)
[프리필터 (Pre-filter)] — 큰 입자(파티클, 스패터) 제거. 세척 가능 금속망 권장
    ↓
[HEPA 필터 (High-Efficiency Particulate Air)] — 0.3µm 입자 99.97% 제거
    ↓
[활성탄 필터 (Activated Carbon)] — VOC, 악취 흡착
    ↓
클린 에어 배출 (실내 재순환 또는 외부 배기)
```

**산업용 유량 기준** (참고치):

- 데스크탑 시스템: 100\~200 m³/h
- 인클로저형 산업용: 250\~400 m³/h
- 대형 생산 셀: 엔지니어링 설계값 적용

> \[인접 도메인: 반도체 클린룸 기술\] 반도체 FAB에서는 ULPA 필터(99.9995% 효율)와 층류 기류 제어로 나노입자를 관리한다. 레이저 가공 집진에서도 초미세 분진 발생 재료(구리, 알루미늄 등) 처리 시 ULPA 적용을 검토할 수 있다.

### 7.3 집진 시스템 고장 증상

| 증상 | 원인 |
| --- | --- |
| 흄 인클로저 밖으로 누출 | 공기 누설 또는 유량 부족 |
| 가공 중 악취 | 활성탄 포화 |
| 가공 품질 저하 (출력 변동) | 광학계 오염 (프리필터 부재) |
| 기류 소음 증가 | 필터 막힘 또는 덕트 폐쇄 |

## 8. 모션 시스템

### 8.1 XY 갠트리 (Gantry)

**구조**: 직선 레일 위를 움직이는 문형(門形) 프레임. 가공 헤드가 XY 평면을 이동.

**특징**:

- 강성 높음, 반복 정밀도 우수 (±0.02\~0.05mm)
- 위치결정 정밀도: ±0.03\~0.07mm (볼스크류 기준)
- 가속도: 최대 1.4G (선형 모터 적용 시 더 높음)
- 대면적 가공에 유리 (수백mm \~ 수m 이상)

**절삭가공 비유**: 브릿지형 CNC 머시닝 센터의 크로스 레일 구조와 동일 개념.

**IPG Photonics 자료**: 갠트리 시스템은 로봇 대비 정밀도·반복성 우수, 저관성 리니어 모션과 이중 구동 랙 적용 시 ±0.03mm 이하 달성.

### 8.2 5축 로봇 (5-Axis Robot)

**특징**:

- 6자유도(DOF)로 3D 복잡 형상·경사면 가공 가능
- 작업 반경 내 어떤 각도든 접근 가능
- 정밀도는 갠트리 대비 낮음 (ARM 구조의 탄성 변형)
- 대형 구조물, 복잡 3D 가공에 유리

**갠트리 vs 로봇 선택 기준**:

| 기준 | 갠트리 선택 | 로봇 선택 |
| --- | --- | --- |
| 정밀도 요구 | ±0.05mm 이하 | ±0.1\~0.5mm 수준 |
| 형상 | 평판, 단순 프로파일 | 복잡 3D, 경사, 관 |
| 면적 | 대형 평판 | 중소형 복잡 형상 |
| 비용 | 중고 | 저(범용 로봇 활용 가능) |

### 8.3 갈바노미터 스캐너 (Galvanometer Scanner)

레이저 마킹, 미세 가공, 고속 가공에 사용하는 고속 빔 편향 시스템.

**원리**: 두 개의 작은 미러(X미러, Y미러)가 각각 갈바노미터 모터에 부착. 미러 각도를 μs 단위로 제어하여 빔을 초고속으로 스캔.

**특징**:

- 스캔 속도: 수 m/s \~ 수십 m/s (미러만 움직이므로 관성 극소)
- 가공 영역: 주로 100mm × 100mm \~ 500mm × 500mm (F-theta 렌즈로 평탄화)
- 정밀도: ±수 µm 수준

**F-theta 렌즈**: 갈바노 스캐너와 조합하는 전용 평탄장 집광 렌즈. 스캔 각도가 θ일 때 초점 위치가 f·θ에 비례 → 플랫 필드 확보.

**절삭가공 비유**: 고속 이송 중 공구 경로를 실시간으로 수정하는 것과 유사. 단, 빔 편향이라 관성이 거의 없다.

**갈바노 + 갠트리 조합 (On-the-Fly 가공)**:

- 갠트리가 이동하는 동안 갈바노 스캐너가 세밀한 패턴을 실시간으로 추가 편향
- 고속 마킹·용접에서 생산성 극대화

### 8.4 선형 모터 (Linear Motor) vs 볼스크류 (Ball Screw)

**볼스크류 방식**:

- 회전 서보 모터 → 볼스크류 → 직선 이동 변환
- 최대 속도: \~120 m/min, 가속도 \~1.5G
- 위치 정밀도: 2\~5µm (선형 스케일 별도 설치 시 향상)
- 백래시(Backlash): 프리로드로 최소화 가능, 마모에 따라 증가
- 비용: 저렴, 구조 단순, 유지 용이

**선형 모터 방식**:

- 전자기력으로 직접 직선 운동 발생 (기어·스크류 없음)
- 최대 속도: \~300 m/min, 가속도 \~10G
- 위치 정밀도: 0.1µm 수준 (선형 인코더 피드백)
- 백래시: 완전 제로 (기계적 연결 없음)
- 비용: 2배 이상, 발열 관리 필요

| 항목 | 볼스크류 | 선형 모터 |
| --- | --- | --- |
| 최대 속도 | \~120 m/min | \~300 m/min |
| 가속도 | \~1.5G | \~10G |
| 위치 정밀도 | 2\~5µm | &lt;0.1µm |
| 백래시 | 있음 (프리로드 최소화) | 없음 |
| 비용 | 낮음 | 높음 (2배+) |
| 유지 | 윤활·마모 관리 | 냉각 관리 |
| 적합 용도 | 일반 가공 | 초고속·초정밀 |

> **수치 출처**: 복수의 모션 시스템 제조사(Aerotech, IMTEKMotion, ZhiyingMotor) 데이터를 종합. 구체적 수치는 제품 라인업에 따라 다를 수 있음.

### 8.5 모션 제어와 레이저 동기화 (PSO 상세)

PSO(Position Synchronized Output)는 8.1절에서 개념을 소개했다. 실제 구현 방식:

1. **인코더 피드백 기반**: 선형 인코더가 위치를 실시간 감지 → 컨트롤러가 설정 간격(예: 0.1mm)마다 레이저 Gate 신호 출력
2. **레이저 응답 시간 보상**: 레이저 ON/OFF 지연(수십\~수백 µs)을 고려하여 Gate 신호를 앞당겨 출력

## 9. 레이저-물질 상호작용 기초

### 9.1 광자가 재료에 닿으면 벌어지는 일

빛(광자)이 재료 표면에 입사하면 세 가지 중 하나가 일어난다:

```
입사 빔 (Incident Beam)
    ↓
  재료 표면
  ├── 반사 (Reflection) → 돌아감
  ├── 흡수 (Absorption) → 재료 내에서 열 또는 전자 여기로 변환
  └── 투과 (Transmission) → 재료를 통과 (불투명 금속에서는 거의 없음)
```

**흡수 + 반사 + 투과 = 1** (에너지 보존)

### 9.2 흡수 (Absorption)

**Beer-Lambert 법칙 (Beer-Lambert Law)**: 레이저가 재료 내부로 진행할수록 지수함수적으로 감쇠.

```
I(z) = I₀ × exp(-α × z)
```

- `I(z)`: 깊이 z에서의 강도
- `I₀`: 표면 강도
- `α`: 흡수 계수 (Absorption Coefficient) \[단위: 1/cm\]
- `z`: 재료 표면에서의 깊이

**금속의 흡수 계수**: \~10⁵ cm⁻¹ → 빛이 수십\~수백 nm 깊이에서 대부분 흡수됨.

**침투 깊이 (Skin Depth, 광학적 침투 깊이)**:

```
δ = 1/α
```

금속에서 δ ≈ 10\~100 nm 수준. 즉, 레이저 에너지는 표면 수십\~수백 nm에 집중 침착.

**절삭가공 비유**: 표면만 파고드는 초정밀 표면 절삭과 같다. 깊이 가공은 용융·기화에 의한 재료 제거로 이루어진다.

### 9.3 파장별 재료 흡수율 — 레이저 소스 선택의 핵심

| 재료 | CO₂ (10.6 µm) | 파이버/Nd:YAG (1.06 µm) | UV (0.35 µm 이하) |
| --- | --- | --- | --- |
| 연강 (Mild Steel) | 5\~15% | 30\~40% | \~60% |
| 스테인리스 | 5\~15% | 35\~45% | \~70% |
| 알루미늄 (상온) | \~2% | 5\~15% | \~15\~20% |
| 구리 (상온) | **\~2%** | \~5% | \~40\~60% |
| 유리·아크릴 | **매우 높음 (&gt;90%)** | 낮음(거의 투과) | 중간\~높음 |
| 폴리머/목재 | 높음 | 낮음 | 높음 |

> **핵심 인사이트**: 구리·알루미늄 등 고반사율 금속은 적외선 레이저 흡수율이 낮아 가공 시작(점화) 어려움. 가열 후 흡수율이 급격히 상승하는 비선형 특성 있음.

> **수치 주의**: 상기 값은 상온 평탄 표면 기준. 온도 상승(용융 직전), 표면 조도, 산화막 유무에 따라 흡수율이 크게 변한다. \[출처: Columbia Univ. AML 레이저 교재, PMC 연구 논문, SPIE 논문\]

### 9.4 반사 (Reflection)

**금속의 높은 반사율 문제**: 구리 상온 기준 적외선에서 반사율 \~98%. 레이저 에너지의 대부분이 반사됨.

**해결 방법**:

1. **짧은 파장 사용**: UV 레이저 → 흡수율 증가
2. **표면 처리**: 산화, 조면화(Roughening) → 다중 반사로 실효 흡수율 증가
3. **고출력 집중**: 반사율이 높은 재료도 파워 밀도가 충분히 높으면 용융 개시 → 이후 흡수율 급상승
4. **백반사 차단 (Back Reflection Isolation)**: 파이버 레이저에서 반사광이 광섬유를 타고 역방향으로 진행 → 광원 손상. 패러데이 아이솔레이터(Faraday Isolator)로 차단

### 9.5 열전도 (Heat Conduction)

레이저 빔이 표면에 흡수된 후, 열은 재료 내부로 전도된다.

**열확산 길이 (Thermal Diffusion Length, L_diff)**:

```
L_diff ≈ √(2 × κ × t_pulse)
```

- `κ`: 열확산율 (Thermal Diffusivity) \[m²/s\]
- `t_pulse`: 레이저 펄스 폭

**펄스 폭과 열 영향 관계**:

| 펄스 폭 | 열확산 길이 (스틸 기준) | 가공 특성 |
| --- | --- | --- |
| ms (밀리초) | \~수십 µm\~mm | 넓은 HAZ, 깊은 열 침투 |
| µs (마이크로초) | \~수 µm | 중간 수준 |
| ns (나노초) | \~수백 nm\~수 µm | 좁은 HAZ |
| ps (피코초) | \~수십 nm | 극히 좁은 HAZ |
| fs (펨토초) | \~수 nm 이하 | 사실상 비열적 가공 가능 |

**절삭가공 비유**: 이송속도·절삭 깊이를 줄이면 열 발생 감소. 레이저에서는 펄스 폭을 줄이면 열 확산 감소.

### 9.6 열영향부 (HAZ — Heat Affected Zone)

**정의**: 레이저 빔에 직접 조사되지 않았지만, 열 전도로 인해 재료 조직이 변화한 영역.

HAZ는 미세구조 변화(마르텐사이트 변태, 과시효, 재결정 등)를 수반하며 기계적 성질을 변화시킨다.

**HAZ 최소화 전략**:

1. 짧은 펄스 폭 (ns → ps → fs 순서로 감소)
2. 낮은 열입력 (낮은 출력 × 빠른 속도)
3. 어시스트 가스로 용융물 신속 제거
4. 인터패스 냉각 (다중 패스 가공 시)

### 9.7 용융 → 기화 → 플라즈마 단계

레이저 강도(Intensity, W/cm²)가 증가함에 따라 재료는 단계적으로 반응한다:

```
낮은 강도 → 가열 (온도 상승)
         → 용융 (Melting): 상변화, 액상 형성
         → 기화 (Vaporization): 증기 발생
         → 플라즈마 (Plasma): 증기가 이온화, 레이저 차폐 가능
```

**플라즈마 차폐 (Plasma Shielding)**:

- 매우 높은 강도에서 플라즈마가 레이저 빔을 흡수·산란
- 재료에 전달되는 실효 에너지 감소
- 특히 ns 이하 펄스나 매우 높은 CW 파워에서 발생
- 천공(Drilling) 깊이 한계의 원인 중 하나

### 9.8 키홀 (Keyhole) 형성

**정의**: 레이저 빔의 높은 강도(통상 10⁶ W/cm² 이상)에서 재료가 국소적으로 기화하면서 형성되는 깊고 좁은 증기 구멍.

**형성 메커니즘**:

1. 레이저가 용융지를 형성
2. 용융지 중앙에서 기화 압력 발생
3. 기화 반발력이 용융물을 외곽으로 밀어냄
4. 깊은 구멍(키홀)이 형성되어 내벽에서 다중 반사 → 흡수율 급상승
5. 레이저가 키홀을 통해 깊이 침투

**키홀 용접(Keyhole Welding)**: 깊고 좁은 용입 비드 형성 가능. 구조물 용접에 유리.

**키홀 불안정과 기공(Porosity)**:

- 키홀이 불안정하게 흔들리거나 갑자기 붕괴되면 용융지에 기공 형성
- 기공은 용접 결함의 주요 원인

### 9.9 콜드 가공 (Cold Processing) — 초단펄스의 비열적 가공

**원리**: 펨토초(fs) 레이저 펄스는 지속 시간이 10⁻¹⁵초 수준. 이 짧은 시간 동안:

1. 광자가 전자에 에너지를 순간 전달 → 전자계가 급격히 가열
2. 전자-격자 에너지 전달 (Electron-Phonon Coupling)이 일어나기 전에 재료가 직접 제거됨
3. 주변 재료에 열이 전도될 시간이 없음 → 비열적(Athermal) 어블레이션

**결과**:

- HAZ 극소화 또는 사실상 없음
- 재주조층(Recast Layer) 없음
- 주변 재료 손상 없음

**한계**:

- 평균 출력이 낮으므로 가공 속도(재료 제거율) 느림
- 시스템 비용 고가 (발진기 + CPA 증폭기)
- 고속 반복 시 누적 열효과(Incubation, Heat Accumulation) 발생 가능

> \[이질 도메인: 정밀 의료/치과 가공\] 임플란트나 치과용 세라믹 가공에서 fs 레이저가 사용되는 이유가 바로 비열적 가공 특성. 주변 조직(금속/세라믹)에 열변성이 없어야 하는 요구사항이 동일.

### 9.10 플루언스 (Fluence)와 어블레이션 임계값 (Ablation Threshold)

**플루언스 (Fluence, F)**: 단위 면적당 에너지 \[J/cm²\] = 강도 × 펄스 폭

**어블레이션 임계값 (F_th)**: 재료 표면에서 실질적인 재료 제거가 시작되는 최소 플루언스.

**금속 어블레이션 임계값 (fs 레이저 기준)** \[출처: SPIE 4423\]:

| 재료 | 단펄스 임계값 (F_th) |
| --- | --- |
| 일반 금속 (평균) | \~0.1 J/cm² |
| Au (금) | \~0.09\~0.1 J/cm² |
| Cu (구리) | \~0.2 J/cm² |
| AISI 316L (SUS) | \~0.1 J/cm² |

> **수치 주의**: 위 값은 fs 레이저 단일 펄스 기준. 다음 조건에서 크게 달라진다:
>
> - **펄스 폭**: ns 레이저는 \~1 J/cm² 이상
> - **누적 효과(Incubation)**: 반복 펄스가 표면을 개질하면 임계값이 낮아짐
> - **파장**: 짧은 파장 → 일반적으로 낮은 임계값
> - **재료 상태**: 표면 산화, 조도, 온도

**어블레이션 깊이 vs 플루언스** (Beer-Lambert 유사 관계):

- 저플루언스 영역: 어블레이션 깊이 ≈ (1/α) × ln(F/F_th) — 광학 침투 깊이 지배
- 고플루언스 영역: 어블레이션 깊이 ≈ (1/α\_eff) × ln(F/F_th) — 열확산 지배

## 10. 가공 공정별 핵심 개념

### 10.1 절단 (Cutting)

**용융 절단 (Melt Cutting)**:

- 재료를 용융 → 어시스트 가스가 용융물 제거
- 낮은 파워·높은 속도 조합 또는 N₂ 가스 사용 시
- 단면: 깨끗하나 얇은 재접착층 가능

**승화 절단 (Sublimation Cutting)**:

- 재료를 직접 기화 → 용융물 없음
- 높은 파워 밀도, 짧은 펄스 필요
- 단면 품질 최고, 재료 제거율 낮음
- 폴리머, 박막 재료에 주로 사용

**산화 절단 (Oxidation/Reactive Cutting)**:

- O₂ 가스와 발열 반응 → 추가 에너지 공급
- 연강 두꺼운 판에 적합 (레이저 단독 에너지 부족한 경우)
- 단면 산화막 형성, 후처리 필요할 수 있음

**절단 파라미터 최적화 목표**: 드로스(Dross) 없음, 직선 줄(Striation) 최소화, HAZ 최소화.

| 불량 유형 | 외관 | 원인 |
| --- | --- | --- |
| 고속 드로스 (Icicle 버) | 하단 날카로운 침상 | 속도 과다 (에너지 부족) |
| 저속 드로스 (글로불러) | 하단 둥근 용탕 덩어리 | 속도 부족 (과열) |
| 줄(Striation) | 단면 물결 패턴 | 속도·파워 불균형, 가스 유동 불안정 |
| HAZ 확대 | 단면 변색, 경도 변화 | 속도 느림, 파워 과다 |

### 10.2 용접 (Welding)

**전도 용접 (Conduction Welding)**:

- 강도: \~0.5 MW/cm² 이하
- 빔이 표면에서 흡수 → 열 전도로 용접지 형성
- 얕고 넓은 비드 (깊이/폭 비 &lt; 1)
- HAZ 적음, 기공 위험 낮음
- 박판, 미관이 중요한 부품(전자, 배터리)에 적합

**키홀 용접 (Keyhole Welding)**:

- 강도: &gt;1\~1.5 MW/cm² 이상
- 키홀 형성으로 깊고 좁은 용입
- 깊이/폭 비 &gt; 3\~10 가능
- 기공, 스패터, 크랙 위험 증가
- 구조물, 두꺼운 부품의 단일 패스 용접에 유리

| 비교 항목 | 전도 용접 | 키홀 용접 |
| --- | --- | --- |
| 파워 밀도 | &lt;0.5 MW/cm² | &gt;1.5 MW/cm² |
| 비드 형상 | 얕고 넓음 | 깊고 좁음 |
| 기공 위험 | 낮음 | 높음 (키홀 붕괴) |
| 스패터 | 적음 | 많음 |
| 적합 재료 | 박판, 정밀 부품 | 두꺼운 구조물 |

### 10.3 마킹/각인 (Marking/Engraving)

**어닐링 마킹 (Annealing Marking)**:

- 표면을 산화 → 색상 변화 (스테인리스에 검정·금·갈색 등)
- 재료 제거 없음 → 표면 평활도 유지
- 의료기기, 식품 용기 등 표면 완전성이 중요한 용도
- 주로 파이버 레이저

**어블레이션 마킹 (Ablation Marking)**:

- 표면 재료 제거 → 음각 형성
- 고대비, 영구적
- 깊이 제어로 각인 깊이 설정

**탄소 이동 마킹 (Carbon Migration)**:

- 탄소 함유 금속(스틸, 스테인리스)에서 표면으로 탄소 이동
- 검은 마크, 빠른 처리
- 항공, 보석 분야

**포밍 마킹 (Foaming)**:

- 플라스틱 표면 용융·거품화 → 밝은 색 돌출 마크
- 금속에는 적용 불가

### 10.4 레이저 클래딩/적층 (Cladding/DED)

**DED (Directed Energy Deposition, 지향성 에너지 적층)**: 용융지에 파우더 또는 와이어를 동시에 공급하며 층층이 적층.

**파우더 피딩 (Powder Feed)**:

- 동축 노즐 또는 다중 노즐로 파우더를 용융지 중심에 공급
- 정밀 적층 가능, 다중 재료 가능
- 재료 이용률: 일반적으로 60\~80% (EHLA 방식: 95% 이상)

**와이어 피딩 (Wire Feed)**:

- 레이저로 와이어를 용융하며 적층
- 높은 적층 속도, 재료 이용률 높음
- 파우더 대비 제어 정밀도 낮음

**주요 파라미터**: 레이저 파워(수 kW), 피딩 속도(수십 g/min), 스캔 속도, 레이어 두께, 오버랩률.

**적용**: 터빈 블레이드 보수, 금형 수리, 내마모 코팅.

### 10.5 드릴링 (Drilling)

**타공 (Percussion Drilling)**:

- 고정된 빔으로 단발 또는 다발 펄스를 집중 조사
- 단순하고 빠름
- 테이퍼(Taper) 발생: 입구 직경 &gt; 출구 직경

**트레파닝 (Trepanning)**:

- 빔이 원형 경로를 따라 이동하며 홀 확장
- 테이퍼 개선, 진원도 향상
- 고종횡비(Aspect Ratio) 홀에 유리

**홀 품질 평가 기준**: 테이퍼(Taper), 진원도(Roundness), 재주조층(Recast Layer) 두께, HAZ.

> **수치 인용**: 타공 대비 트레파닝은 고종횡비(&gt;10:1) 홀에서 기하학적 품질 우수 \[TWI Global 자료\]. 최적 조건: 타공 200A·NaCl 냉매 조합에서 테이퍼·진원도 최소화 확인 (연구논문 기준, 실제 산업 조건과 다를 수 있음).

### 10.6 표면처리 (Surface Treatment)

- **레이저 열처리 (Heat Treatment)**: 표면만 집중 가열 → 마르텐사이트 변태 → 표면 경화. 변형 최소.
- **텍스처링 (Texturing)**: 표면에 마이크로/나노 구조 형성 → 마찰계수, 젖음성, 광학 특성 변경.
- **세정 (Cleaning)**: 산화막, 코팅, 오염물을 어블레이션으로 제거. 도장 전처리, 용접부 세정.

## 11. 레이저 안전

### 11.1 레이저 등급 분류 (IEC 60825-1)

**국제 표준**: IEC 60825-1 (국내: KS C IEC 60825-1). 북미: ANSI Z136.

| 등급 | 파워 수준 | 주요 위험 | 필요 조치 |
| --- | --- | --- | --- |
| **Class 1** | 매우 낮음 | 정상 사용 시 안전, 밀폐형 | 없음 (밀폐 조건 유지) |
| **Class 1M** | Class 1 수준 | 광학기기 사용 시 위험 | 광학기기 접근 금지 |
| **Class 2** | &lt;1 mW (가시광) | 장시간 직시 위험 (반사로 보호) | 직시 금지 |
| **Class 3R** | 1\~5 mW | 직시 즉시 위험 | 경고 표시 |
| **Class 3B** | 5\~500 mW | 직접·경면 반사 위험, 피부 위험 | **보호 고글 필수**, 인터록 |
| **Class 4** | &gt;500 mW | 직접·산란 반사 모두 위험, 화재 | **보호 고글, 인클로저, 인터록, 교육** |

> **실무 핵심**: 산업용 레이저 절단기(수 kW)는 모두 Class 4 소스를 사용한다. 단, 완전 밀폐 인클로저 + 인터록으로 Class 1 수준의 완제품으로 분류하는 것이 일반적.

### 11.2 주요 위험 요소

1. **직접 빔 (Direct Beam)**: 초고강도. 눈 망막 즉각 손상, 피부 화상.
2. **경면 반사 (Specular Reflection)**: 금속 표면 거울 반사. 직접 빔과 유사한 강도.
3. **산란 반사 (Diffuse Reflection)**: 거친 표면에서 산란. Class 4에서는 산란광도 위험.
4. **흄과 파티클**: 호흡기, 피부 자극. 독성 금속 산화물(크롬, 니켈 등).
5. **고전압**: 전원 공급 장치, 특히 CO₂/Nd:YAG 플래시램프 회로에 수 kV.
6. **화재**: Class 4 레이저의 발화 위험. 가연성 재료 근처 주의.

### 11.3 보호 장비 (PPE)

**레이저 보호 고글**:

- 파장과 광학 밀도(OD, Optical Density) 기준에 맞는 것 선택
- OD = log₁₀(입사 강도/투과 강도). OD 4 → 10,000배 감쇠
- **반드시 해당 레이저 파장 표시 확인**: 파이버 레이저용(1064nm)과 CO₂용(10.6µm)은 다름

**인클로저 창**:

- 재료에 따라 해당 파장 차단 성능 다름
- 파이버 레이저: 특수 코팅 유리 또는 OD≥5 필터 패널
- CO₂ 레이저: 아크릴(PMMA)은 10.6µm 차단 → CO₂ 가공 창문으로 사용 가능

### 11.4 안전 관리 실무

- **LOTO (Lockout-Tagout)**: 정비·청소 시 레이저 전원 차단·잠금
- **경고등**: 레이저 동작 중 황색/적색 경고등 점등
- **출입 통제**: 가동 중 인클로저 개방 시 빔 자동 차단 (인터록 연동)
- **비상 정지 (E-Stop)**: 즉시 접근 가능한 위치에 설치
- **안전 책임자**: 대형 설비는 Laser Safety Officer(LSO) 지정 권장

## 12. 사양서 (Spec Sheet) 읽는 법

레이저 소스 데이터시트의 각 항목이 가공에 무엇을 의미하는지 연결한다.

### 12.1 주요 스펙 항목 해설

**정격 출력 (Rated Power)**

- 단위: W 또는 kW
- **가공 의미**: 최대 가공 가능 두께, 절단 속도의 상한.
- 단, 정격 출력으로 지속 가공 시 열적 스트레스 증가. 실제 운용은 70\~90% 수준.

**파장 (Wavelength)**

- 단위: nm 또는 µm
- **가공 의미**: 재료별 흡수율 결정. 파장에 맞는 광학 부품 필수. ([9.3절](#93-%ED%8C%8C%EC%9E%A5%EB%B3%84-%EC%9E%AC%EB%A3%8C-%ED%9D%A1%EC%88%98%EC%9C%A8--%EB%A0%88%EC%9D%B4%EC%A0%80-%EC%86%8C%EC%8A%A4-%EC%84%A0%ED%83%9D%EC%9D%98-%ED%95%B5%EC%8B%AC) 참조)
- 파이버 레이저: 1070\~1080 nm
- CO₂: 10.6 µm (또는 9.3 µm)
- Nd:YAG: 1064 nm
- 그린 레이저: 532 nm
- UV 레이저: 355 nm

**M² (빔 품질 인수, Beam Quality Factor)**

- 정의: 실제 빔이 이상적인 가우시안 빔(M²=1)으로부터 벗어난 정도. 낮을수록 좋음.
- 단일 모드(SM): M² ≈ 1.05\~1.2
- 멀티 모드(MM): M² &gt; 1 (수십 이상 가능)
- **가공 의미**: M²가 낮을수록 → 더 작은 초점 스폿 → 더 높은 파워 밀도 → 더 얇은 절단 폭(Kerf), 더 정밀한 가공. M² × 2배 증가 → 초점 스폿 직경 × 2배 → 파워 밀도 × 1/4.

**BPP (Beam Parameter Product, 빔 파라미터 곱)**

- 정의: 빔 허리(Beam Waist) 반경 × 반발산각 \[mm·mrad\]
- 관계: BPP = M² × λ/π
- **가공 의미**: BPP가 작을수록 더 작게 집속 가능. 광섬유 코어 직경이 작아질수록 BPP 유리.

**파이버 코어 직경 (Fiber Core Diameter)**

- 단위: µm (마이크로미터)
- 일반 범위: 50\~600 µm
- **가공 의미**: 코어 직경이 작을수록 더 작은 집속 스폿 가능 (BPP 연결). 절단용 고출력 파이버 레이저는 100\~200 µm 코어 일반적. 정밀 가공용 소출력은 50 µm 이하.
- **커넥터 타입**: QD, QBH, LLK 등 제조사별 표준. 가공 헤드와 호환성 확인 필수.

**발산각 (Divergence)**

- 단위: mrad
- **가공 의미**: 콜리메이터\~집광 렌즈 간 거리(BFL, Back Focal Length)와 함께 최소 스폿 크기 결정.

**펄스 파라미터** (펄스 레이저 해당):

- 펄스 에너지 (Pulse Energy) \[J 또는 mJ\]
- 펄스 폭 (Pulse Width/Duration) \[ns, ps, fs\]
- 반복률 (Repetition Rate) \[Hz, kHz\]
- 피크 파워 (Peak Power) \[kW, MW, GW\] = 펄스 에너지 / 펄스 폭

**냉각 요구사항 (Cooling Requirements)**

- 냉각 유량, 입출구 온도, 압력 범위
- **실무 연결**: 칠러 사양 선정의 직접 입력값.

**전력 소비 (Power Consumption)**

- 단위: kW 또는 kVA
- **실무 연결**: 설치 전기 용량 계획.

**동작 온도 (Operating Temperature)**

- 통상 10\~40°C (±5°C 수준)
- **실무 연결**: 공장 환경 조건 확인. 고온·다습 환경은 별도 공조 검토.

### 12.2 스펙 읽기 예시 — 3kW 파이버 레이저

```
사양 예시 (가상)
  출력: 3000 W
  파장: 1070 ± 10 nm
  M²: <1.1 (SM), <2.5 (MM 옵션)
  BPP: <0.4 mm·mrad (SM)
  파이버 코어: 100 µm / QBH 커넥터
  냉각수: 유량 10 L/min, 온도 20°C ±1°C
  전력: 8 kVA
  인터페이스: EtherCAT + 아날로그 0~10V
```

**해석**:

- M²&lt;1.1 → 이상 가우시안에 가까운 우수한 빔 품질 → 미세 가공/고속 절단 가능
- 100 µm 파이버 → 중간 수준 스폿 크기 → 범용 절단 헤드와 매칭 가능
- EtherCAT 인터페이스 → PSO 지원 가능, 실시간 파워 제어 가능
- 냉각수 온도 ±1°C → 칠러의 정밀도 요구사항 확인 필요

## 13. 트러블슈팅 기초

### 13.1 출력 저하 (Power Drop)

| 원인 | 진단 방법 | 조치 |
| --- | --- | --- |
| 광학 오염 (Optical Contamination) | 렌즈·미러 육안 또는 UV 조사 검사 | 규정 클리닝 → 심각 시 교체 |
| 정렬 이탈 (Misalignment) | 빔 프로파일러 또는 번-페이퍼 | 광학 정렬 재조정 |
| 레이저 소스 열화 | 소스 직결 파워미터 측정 | 소스 정비/교체 |
| 냉각수 온도 이탈 | 칠러 제어판 모니터링 | 냉각수 교체·칠러 점검 |
| 파이버 손상 | 파이버 단면 검사 | 파이버 교체 |

### 13.2 빔 품질 변화 (Beam Quality Degradation)

**열렌즈 (Thermal Lensing)**:

- 광학 부품이 레이저를 부분 흡수 → 가열 → 굴절률 변화 → 렌즈 효과 발생
- 증상: 가동 초기와 정상 가동 중 초점 위치가 다름 (워밍업 후 안정)
- 조치: 고순도 광학 부품 사용, 적절한 냉각, 오염 방지

**모드 불안정 (Mode Instability)**:

- 고출력 파이버 레이저에서 특정 출력 이상에서 갑자기 빔 프로파일 변동
- 증상: 가공 품질 갑작스러운 저하, 출력 진동
- 조치: 파이버 레이저 제조사에 상담, 출력 한계 이하 운용

### 13.3 가공 불량과 레이저 원인

| 불량 현상 | 가능한 레이저 원인 | 확인 방법 |
| --- | --- | --- |
| 절단 버(Burr) 과다 | 파워 부족, 속도 과다, 가스 압력 부족 | 파워미터 확인, 파라미터 조정 |
| 드로스 (Dross) | 파워 과다 또는 부족, 가스 조건 불량 | [10.1절 표](#101-%EC%A0%88%EB%8B%A8-cutting) 참조 |
| HAZ 확대 | 파워 과다, 속도 느림, 냉각 불충분 | 단면 검사, 파라미터 조정 |
| 불완전 용입 | 파워 부족, 빔 품질 저하, 초점 이탈 | 파워 확인, 초점 재조정 |
| 마킹 불균일 | 갈바노 스캐너 정렬 이탈, F-theta 렌즈 오염 | 교정 패턴 마킹 테스트 |
| 펄스 불안정 | 제어기 신호 노이즈, 레이저 소스 불량 | 오실로스코프로 Gate 신호 확인 |

## 14. 연구자 메모

### 출처 평가

- **흡수율 수치 (9.3절)**: 복수 출처에서 인용. 상온·평탄면 기준이며 실제 가공 조건과 차이 있을 수 있음. \[인접 도메인: 광학 측정 연구\] — 가공 현장에서 표면 조도·온도가 다르므로 수치 편차 예상.
- **어블레이션 임계값 (9.10절)**: fs 레이저 단일 펄스 기준. ns 레이저에서는 수배 높음. 실제 가공에서는 누적 효과로 더 낮아질 수 있음.
- **모션 사양 수치 (8.4절)**: 제조사 카탈로그 기반. 실제 응용에서는 하중, 스트로크, 온도에 따라 차이 있음.
- **냉각수 전도도 요구 (3.2절)**: 제조사별 상이. 구체적 사양은 레이저 소스 매뉴얼 확인 필수.

### 반증 탐색

- **fs 레이저 = 무조건 고품질**: 반증 있음. 고속 반복 시 열 누적(Heat Accumulation)으로 HAZ 발생 가능. "콜드 가공"은 저반복률·저파워 조건에서만 성립.
- **갠트리가 로봇보다 항상 정밀**: 반증 있음. 복잡 3D 형상·접근성에서는 로봇이 필수. 정밀 로봇은 ±0.02mm 수준도 가능.
- **키홀 용접 = 기공 필연**: 반증 있음. 파라미터 최적화, 진공 환경, 레이저 파워 변조로 기공 크게 감소 가능.

### 실행 연결

이 보고서의 정보로 가능한 의사결정:

1. **소재별 레이저 소스 선택**: 구리·황동 → 파이버 레이저 단파장(그린/UV) 검토, CO₂ 부적합
2. **집진 시스템 사양**: 재료별 독성 분진 특성 고려 (스테인리스 → 크롬 Cr(VI) 분진 HEPA 필수)
3. **냉각수 관리 주기 설정**: 3개월 교체, 6개월 배관 세정
4. **PSO 기능 유무 확인**: 고속 마킹/절단 시 PSO 없으면 코너 과열 발생 — 컨트롤러 사양 필수 확인
5. **M² 값으로 가공 한계 예측**: M²×파장으로 최소 초점 스폿 계산 후 요구 정밀도와 비교

### 관점 확장

**결론을 바꿀 수 있는 인접 질문**:

1. **고출력 레이저의 모드 불안정 임계값**: 파이버 레이저는 \~수백 W 이상에서 TMI(Transverse Mode Instability) 발생 가능. 특정 파이버 설계에서는 이 임계값이 가공 조건과 겹침 → 빔 품질 예측 불가.
2. **어시스트 가스 가격 변동**: 고순도 N₂·Ar의 가격 상승이 가공 비용에 직접 영향. 압축 공기 활용 범위 확대가 최근 트렌드.

**숨은 변수**:

- **레이저 소스 노화**: 파이버 레이저의 펌프 다이오드는 수만 시간 수명. 노화에 따른 점진적 출력 저하를 주기 점검으로 조기 감지해야 함.
- **주변 온도·습도**: 광학 부품의 열팽창, 결로 → 정밀 가공 기계에서 환경 제어(공조) 중요.
