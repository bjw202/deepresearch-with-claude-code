## Executive Summary

초단파 펄스(USP) 레이저 — 펨토초(fs, 10⁻¹⁵s)와 피코초(ps, 10⁻¹²s) — 는 "콜드 어블레이션"이라는 고유한 물리적 메커니즘을 통해 **산업 제조, 의료, 바이오, 기초과학** 전 분야에 걸쳐 기존 공정으로 불가능했던 정밀 가공·치료·측정을 가능하게 하고 있다. 2025년 기준 전 세계 시장 규모는 약 **$2.4~2.9B**이며 **CAGR 15~17%**(2025~2031)로 고성장 중이다.

**핵심 결론 3가지:**

1. **"fs가 항상 ps보다 우월하다"는 통념은 틀렸다.** 어플리케이션별 정밀도·속도·비용 요구에 따라 최적 펄스폭이 다르며, 10~50 ps 경계에서는 금속 가공 품질 차이가 미미하여 ps가 비용 합리적이다. 유리 등 투명 유전체에서는 ps도 fs와 유사한 냉간 절제를 달성한다.

2. **GHz 버스트 모드와 kW급 고출력화가 게임 체인저이다.** BiBurst 모드로 Si 어블레이션 속도 23배 향상, TRUMPF 1kW USP 레이저(2024) 출시로 EV 배터리·반도체 양산 공정 진입이 가속화되고 있다.

3. **기술적 우월성보다 TCO(총소유비용) 관점이 채택을 결정한다.** SiC 슬라이싱 재료 손실 저감, 유리 필라멘테이션 단일패스 공정, 배터리 전극 버 제거 등 시스템 수준의 경제성이 실제 채택을 이끌고 있다.

## 2. 펨토초 vs 피코초: 통합 선택 가이드

> **Critic 권고 반영**: R3의 HAZ "<1µm" 수치는 이상적 조건의 일반론이며, R1의 실측치(Al 기준 ≤1.5µm)를 기준으로 통일. "15-30% 속도 차이"는 제조사 주장으로 태깅.

### 2.1 물리적 차이 (실측 기반)

| 지표 | 펨토초 (fs) | 피코초 (ps) | 출처 |
|------|-----------|-----------|------|
| HAZ (Al, 실측) | ≤1.5 µm | 2~5 µm | MDPI Micromachines 2014 |
| HAZ (ns 참조) | — | — | 20 µm (8ns) → 13배 차이 |
| Ra (금속) | 0.02~0.1 µm | 0.1~0.5 µm | 복수 학술 출처 |
| 처리속도 | 기준 | 15~30% 빠름 (제조사 주장) | opmtlaser.com |
| 초기투자비 | 매우 높음 | 높음 (fs 대비 저렴) | 복수 출처 |

### 2.2 재료별 경계 조건

- **금속**: 10ps 이하에서 fs와 ps의 품질 차이 미미 → 비용 합리적인 ps 선택 가능
- **투명 유전체(유리)**: 열 확산 시간 50-100ps 초과 → ps도 fs와 유사한 냉간 절제 달성
- **생체조직**: 열 확산 ~1µs이나 정밀도 요구 → fs 필수 (안과, 신경외과)

### 2.3 의사결정 매트릭스 (통합)

| 요구사항 | → fs | → ps |
|---------|------|------|
| HAZ <1µm 필수 | ✅ | ❌ |
| Ra <0.1µm 필수 | ✅ | ❌ |
| 투명 소재 내부 가공 | ✅ | △ (일부 가능) |
| 생체 적합성 유지 필수 | ✅ | ❌ |
| 처리량 우선 | ❌ | ✅ |
| 비용 효율 우선 | ❌ | ✅ |
| 유리 절단 (양산) | △ | ✅ (OLED 표준) |
| 금속 마킹/텍스처링 | △ | ✅ |

## 6. 누락 관점 보완 (Critic 지적 반영)

### 6.1 환경·안전·규제
- USP 레이저는 **Class 4** 레이저 — 인터록, 차폐, 보호안경 필수
- 가공 시 발생하는 **금속 나노입자/퓸(fume)**: 작업자 건강 이슈, 배기·필터 시스템 필수
- 의료 분야 규제 승인 경로가 어플리케이션마다 다르며, 신규 어플리케이션(뇌전증 등)은 5-10년 소요 전망

### 6.2 실패 모드 (보고서에서 누락된 핵심)
- **유리 필라멘테이션**: 크랙 전파 방향 이탈, 강화유리 잔류응력으로 비정상 파단
- **배터리 전극 패터닝**: 채널 깊이 불균일 → 전해액 편류, 국부 리튬 도금 위험
- **고반복률(>1MHz) 열 누적**: "콜드 어블레이션" 가정 붕괴, ΔT ~7000°C 가능

### 6.3 경쟁 기술
- **DRIE(Deep Reactive Ion Etching)**: MEMS에서 USP 레이저와 경쟁. 대량 양산·높은 종횡비에서 DRIE 우위
- **EDM(방전가공)**: 금형·미세홀에서 여전히 비용 경쟁력 보유
- **나노초 레이저**: 마킹·조각·거친 절삭에서 TCO 우위 유지

## 7. 역방향 의사결정 가이드

> "결과가 X이면 → Y를 조정하라"

| 만약... | 그러면... |
|---------|----------|
| 공차 요구가 ±0.01mm 이하이고 HAZ <5µm 필수 | ns 레이저 대신 USP(ps 이상) 도입을 검토하라 |
| 공차 ±0.005mm 이하, Ra <0.1µm 필요 | ps가 아닌 fs를 선택하라 |
| 유리/사파이어 절단이고 양산 처리량이 우선 | fs 대신 ps(Bessel 빔 + 필라멘테이션)를 선택하라 |
| 배터리 전극 패터닝을 검토 중 | 벤치스케일 성과(NREL)를 양산 roll-to-roll에서 재검증하라 |
| 반복률 >1MHz 운용 예정 | 열 누적 모델링을 선행하라. "콜드 어블레이션" 가정이 깨질 수 있다 |
| 중국산 USP 레이저를 검토 중 | 빔 품질(M²), MTBF, 서비스 인프라를 서구 제품과 직접 비교하라 |
| GHz 버스트 모드 도입 검토 | 재료 의존성이 강하므로 대상 재료에서 반드시 사전 테스트하라 |
| 투자 회수 기간이 3년 초과 전망 | ps로 다운그레이드하거나 기존 ns/기계 공정을 유지하라 |

## 8. 예상 밖 핵심 발견

1. **유리 용접의 산업화 진전**: USP 레이저로 유리-금속 직접 접합이 가능해지며, 접합 강도가 모재를 초과한다. 접착제·패스너 없는 유리 밀봉이 센서·의료기기·전자패키징에 새로운 설계 자유도를 제공한다.

2. **배터리 전극 레이저 패터닝의 비용 충격**: NREL 데이터에 따르면 kWh당 $1.3~1.5 추가로 4C 충전 성능을 2배 향상시킬 수 있다. 이는 EV 충전 인프라 비용 대비 극히 작은 투자이다. (단, 양산 검증 필요 ★★)

3. **아토초 과학→산업 파이프라인**: 2023 노벨상 수상 기술인 HHG 기반 XUV 광원이 반도체 메트롤로지·EUV 마스크 검사에 활용되기 시작. 기초과학→산업 전환이 예상보다 빠르게 진행 중.

4. **양자컴퓨팅 레이저 서브시스템**: NKT-IonQ 파트너십(2024)으로 USP 레이저가 양자컴퓨터 핵심 부품으로 부상. 새로운 고부가 시장 형성 중.
