## 핵심 결론 요약

1. **표준화된 단일 "강건성 지수"는 아직 존재하지 않는다.** 산업 표준(MIL-PRF-8625, QUALANOD)은 개별 항목별 Go/No-Go 판정만 제공하며, 통합 지수는 학계에서도 공식 발표된 것이 없다.

2. **EIS 기반 정량 지표가 가장 유망한 기반이나, 단일 지표의 한계가 명확히 입증되었다.** |Z|₀.₁Hz는 barrier 성능만 반영하고 scribe 부식을 예측하지 못하며(Piccinini 2023), f_b는 CPE 비이상성에 취약하고, Brasher-Kingsbury는 CPE 보정 없이 과대추정한다.

3. **시간 적분 지표(∫(1/|Z|)dt)가 가장 유망하나 검증이 부족하다.** Bongiorno et al.(2024)의 단일 연구에 기반하며, 유기 코팅에서만 검증됨. 아노다이징 피막에 대한 직접 검증은 미수행. ~~[Critic 이슈 5-1 반영: "가장 강건" → "가장 유망하나 검증 부족"으로 하향]~~

4. **제조 현장 적용을 위해서는 계층적(Tiered) 접근이 현실적이다.** 인라인 전수검사(와전류+어드미턴스) + 오프라인 로트검사(EIS+접착력) + 정기 가속시험의 3단계 구조.

5. **봉공 방식과 합금 종류가 지수 설계의 핵심 층화 변수이다.** [Critic 이슈 4-2, 4-3 반영] 동일 |Z| 또는 어드미턴스 값이라도 봉공 방식(열수/냉간/니켈)과 합금(6xxx/2xxx/7xxx)에 따라 의미가 다르므로, ARI 지수에 봉공 방식과 합금을 입력 변수로 포함해야 한다.

## 1. EIS 기반 지표 6종 + 2종(신규) 비교

| 지표 | 수식 | 장점 | 한계 | 도메인 적용성 | 실용성 |
|------|------|------|------|--------------|--------|
| **\|Z\|₀.₁Hz** | 0.1Hz 임피던스 크기 | 측정 간편, 등급 기준 확립 | barrier만 반영, scribe 부식 예측 불가 [Piccinini 2023] | 유기 코팅 기준을 아노다이징에 정성적 적용 가능 (Al 6061 검증) | ★★★★★ |
| **PE%** | (Rp,coated - Rp,bare)/Rp,coated × 100 | bare 대비 직관적 비교 | 기준 시편 필요, 합금/전해질 의존 | 유기 코팅 및 아노다이징 모두 적용 | ★★★☆☆ |
| **P%** (신규) | Rp,bare/Rp,coated | 전기화학적 활성 면적 비율 추정 | 기하학적 기공률과 다름 | PVD 코팅에서 개발, 아노다이징 적용 가능 | ★★☆☆☆ |
| **CDI%** | (Zi - Zf)/Zi \|₀.₁Hz × 100 | 열화 추적에 적합 | 초기값 의존, 고품질 코팅에서 과잉 경보 | 유기 코팅 기반이나 아노다이징 추적에도 유효 | ★★★★☆ |
| **DF%** | Bode plot 면적비 | 전 주파수 대역 활용, 자동화 용이 | 계산 복잡 | 유기 코팅 기반 | ★★☆☆☆ |
| **fb** | 위상각 = -45° 주파수 | 단일 값, 열화 민감 | CPE 비이상성에 취약 [PMC 8358942]; 건전 피막에서 측정 불가 | 유기 코팅 기반, CRPA 대안 존재 | ★★★☆☆ |
| **∫(1/\|Z\|)dt** | 저주파 임피던스 역수 시간 적분 | 이력 통합, 시험 조건 무관 | 연속 모니터링 필요, 아노다이징 미검증 | **[인접 도메인: 유기 코팅]** 아노다이징 직접 검증 미수행 | ★★☆☆☆ |
| **φ (Brasher-Kingsbury)** | log(Ct/C₀)/log(εwater) | 수분 침투 정량 | CPE 보정 필수, 과대추정 경향 [Lacombe 2017] | **[인접 도메인: 유기 코팅]** 아노다이징 다공 구조와 메커니즘 다름 | ★★☆☆☆ |

### Critic 반영 사항
- |Z| 등급 기준(10⁸/10⁶)의 아노다이징 유효성은 Al 6061 단일 연구(Boisier 2008)에만 의존 [이슈 1-1]. 다른 합금계(2xxx, 7xxx)에서의 검증이 부족하며, 확신도를 "정성적으로 유효"에서 "6xxx 계열에서 유효, 다른 합금계는 미검증"으로 조정.
- 등급 분류가 eis-methodology(4단계)와 quantification-methods(3단계)에서 불일치 [이슈 3-1]. 아래 통합 등급 제안:

| |Z|₀.₁Hz (Ω·cm²) | 등급 | 원전 | 비고 |
|---|---|---|---|
| > 10⁹ | 매우 우수 | ISO 16773-2 | 고임피던스 코팅 정의 |
| 10⁸ ~ 10⁹ | 우수 | USBR 8540-2019 | 장기 사양 후보 최소 수준 |
| 10⁶ ~ 10⁸ | 보통 | Bacon 1948 / Amirudin 1995 | 열화 진행 영역 |
| < 10⁶ | 불량 | 문헌 합의 | 피막 보호 기능 상실 |

## 2. 제안: 아노다이징 피막 강건성 지수 (ARI v2)

### 2.1 v1 대비 변경점

| 항목 | v1 | v2 (Critic 반영) |
|------|-----|-----|
| 가중치 근거 | "권장" (근거 미명시) | AHP 기반 초기 설정 → 데이터 축적 후 교정 (미확정 명시) |
| 봉공 방식 | 미고려 | **층화 변수로 포함** |
| 합금 종류 | 미고려 | **입력 변수로 포함** |
| 정규화 기준 | 절대값 | 합금별/봉공별 참조값 사용 |
| 등급 경계값 | 85/70/55 (임의) | Phase 2에서 필드 데이터로 교정 (임의임을 명시) |

### 2.2 Gate 판정 (필수 통과 기준)

| 항목 | 시험법 | 최소 기준 | 근거 | 이 기준이 틀릴 수 있는 조건 |
|------|--------|----------|------|---------------------------|
| 피막 두께 | 와전류 (ASTM B244) | 사양 하한 -10% | MIL-PRF-8625 | 두꺼운 피막은 열충격 크레이징 위험 증가 |
| 봉공 품질 | 어드미턴스 (ISO 2931) | ≤ 20 μS | QUALANOD | 냉간 봉공에 적용 불가; 얇은 피막(<10μm)에서 과소 감지 가능 |
| 접착력 | Cross-cut (ISO 2409) | ≥ 4B | 산업 관행 | 고곡률 형상에서는 3B도 허용 가능 |
| 기본 내식성 | \|Z\|₀.₁Hz | ≥ 10⁶ Ω·cm² | 문헌 합의 (하한) | 6xxx 계열에서 검증. 2xxx/7xxx는 별도 임계값 필요 가능 |

### 2.3 ARI 점수 산출

Gate 통과 후, 아래 항목의 가중합으로 0~100 점수를 산출한다.

```
ARI = w₁·S_eis + w₂·S_seal + w₃·S_adhesion + w₄·S_thickness
```

**가중치 상태**: 현재 미확정. Phase 1에서 AHP 전문가 합의로 초기 설정 예정. v1에서 제안된 0.45/0.25/0.15/0.15는 전문가 판단 기반의 **잠정안**이며, 다음 조건에서 부적절할 수 있다:
- 해양 환경: S_eis 가중치를 더 높여야 할 수 있음
- 소비전자 외관 제품: S_adhesion + 외관 안정성 가중치가 더 중요할 수 있음
- 반도체 장비용: S_eis에 절대적 비중 필요

[Critic 이슈 2-1 반영: 가중치의 근거와 미확정 상태를 명시]

### 2.4 봉공 방식별 층화 [Critic 이슈 4-2 반영]

동일 ARI 산식이라도 봉공 방식에 따라 해석이 달라진다:

| 봉공 방식 | EIS 특성 | 어드미턴스 적용성 | ARI 적용 시 주의 |
|-----------|----------|-------------------|-----------------|
| 열수 (Boiling water) | 높은 Rp, 2-시정수 명확 | ISO 2931 직접 적용 | 표준 적용 가능 |
| 니켈 아세테이트 | 높은 Rp, 열수와 유사 | 어드미턴스 응답 상이, 기준 조정 필요 | Gate 기준 별도 설정 |
| 저온/냉간 (Cold seal) | Rp 변동성 큼, 숙성 필요 | ISO 2931 적용 불가 | 산용해 시험으로 대체 |
| 중온 (Mid-temperature) | 열수와 냉간의 중간 | 부분 적용 가능 | 추가 검증 필요 |

### 2.5 합금별 참조값 [Critic 이슈 4-3 반영]

| 합금 계열 | 주요 합금 | 기대 |Z|₀.₁Hz 범위 | ARI 정규화 참조 | 비고 |
|-----------|----------|---------------------|------------|------|
| 6xxx | 6061, 6063 | 10⁷ ~ 10⁹ Ω·cm² | Boisier 2008 데이터 | 소비전자 주력 합금 |
| 5xxx | 5052, 5005 | 10⁷ ~ 10⁹ Ω·cm² | 유사 범위 예상 | Si/Mn 함량 낮아 어드미턴스 적용 가능 |
| 2xxx | 2024-T3 | 10⁵ ~ 10⁸ Ω·cm² | 별도 연구 필요 | Cu 석출물로 국부전지, 임피던스 하한 하향 |
| 7xxx | 7075-T6 | 10⁶ ~ 10⁸ Ω·cm² | 별도 연구 필요 | Zn/Cu 영향, 피팅 민감 |

## 3. EIS 해석 주관성 해소 방안

### 3.1 피팅-프리(Fitting-free) 지표 우선 사용

|Z|₀.₁Hz와 위상각은 등가회로 피팅 없이 raw 데이터에서 직접 추출. 이것이 가장 강력한 주관성 제거 방법.

### 3.2 AutoEIS/ML 자동화

| 도구 | 성숙도 | 아노다이징 적용성 | 비고 |
|------|--------|------------------|------|
| AutoEIS (JOSS 2025) | 연구용 성숙 | 재훈련 필요 | Bayesian 모델 선택, 불확실성 정량화 |
| CNN 기반 ECM 분류 | 벤치마크 수준 | 아노다이징 데이터셋 없음 | 노이즈 강건성 검증 중 |
| 비지도 클러스터링 (PCA+t-SNE) | 소규모 검증 | **아노다이징 전용 재검증 필수** | per-block normalization 파라미터 튜닝 필요 |

[Critic 이슈 5-2 반영: "적은 커스터마이징으로 적용 가능" → "아노다이징 전용 재검증 필수"로 수정]

## 4. 제조 현장 적용 로드맵

### Phase 1: 기반 구축 (즉시) — 추정 비용 포함 [Critic 이슈 4-1 반영]

```
[인라인]                           [오프라인 로트 검사]
와전류 두께 (전수, $2K~$5K/장비)     EIS |Z|₀.₁Hz (로트당 3~5개, $15K~$50K/포텐시오스탯)
어드미턴스 (N개당 1개, $3K~$8K)      Cross-cut 접착력 (로트당 2~3개, <$1K)
        ↓                                  ↓
    Gate 판정                          ARI 점수 산출
```

- EIS 장비 비용: 기본형 포텐시오스탯 $15K~$30K, 고임피던스(>10⁹ Ω) 대응 $50K~$100K+
- 패러데이 케이지, 전해질 관리 등 부대비용: $5K~$10K
- **비용효율 비교**: 와전류($2K) + 어드미턴스($5K) = $7K로 인라인 QC 즉시 가능. EIS는 오프라인 로트 검사에 한정하면 1대로 충분.

### Phase 2: 상관관계 확립 (3~6개월)

- ARI + 가속시험(염수분무 336h, 인공땀 168h, 85/85 500h) 동시 수행
- ARI 등급 경계값 교정 (현재 85/70/55는 임의값)
- **봉공 방식별, 합금별 층화 분석** 포함

### Phase 3: 지능화 (6~12개월)

- EIS 자동 측정 + AutoEIS 자동 판정 시스템
- SPC 관리도에 ARI 트렌드 표시
- 비지도 ML 파이프라인 시범 도입 (아노다이징 전용 데이터셋 구축 후)

## 5. 동남아 환경 특화 고려사항

### 5.1 환경 대응 Gate 강화

| 항목 | 일반 기준 | 동남아 강화 기준 | 근거 |
|------|----------|----------------|------|
| \|Z\|₀.₁Hz | ≥ 10⁶ Ω·cm² | ≥ 10⁷ Ω·cm² | 고온고습 환경에서 전기화학적 부식 가속 |
| 어드미턴스 | ≤ 20 μS | ≤ 15 μS | 불완전 봉공의 영향이 고습 환경에서 증폭 |
| 인공땀 침지 | - | 168h 후 ΔE* < 3, 피팅 없음 | 높은 발한량에 대한 내성 필수 |
| 85/85 내습 | - | 500h 후 CDI% < 20% | "20년 등가"는 반도체 기준; 아노다이징 가속 계수는 미확인 [이슈 1-3] |

### 5.2 인공땀 시험 프로토콜

ISO 3160-2 기반이나, 소비전자 OEM은 더 가혹한 내부 표준을 사용할 수 있음 [이슈 1-5]. 시계 산업 출처를 그대로 적용하기보다, 소비전자 접촉 조건(면적, 시간, 온도)을 반영한 조정이 필요.

| |Z| 등급 3단계 vs 4단계 | quantification: >10⁸ = Excellent (3단계) | eis-methodology: >10⁹ = 매우 우수 (4단계) | **4단계 채택** | ISO 16773-2가 >10⁹를 별도 정의. 출하 기준에서 "우수(10⁸)"와 "매우 우수(10⁹)"를 구분하는 것이 실무적으로 유용 |

## 9. 불확실성 및 한계

### 높은 확신 (문헌 합의)
- |Z|₀.₁Hz > 10⁸ Ω·cm²이면 우수한 보호 피막 (단, 유기 코팅 기준의 정성적 이식)
- EIS 등가회로 피팅에 주관성이 있으며, 피팅-프리 지표 우선 사용이 타당
- 아노다이징 피막은 2-시정수 등가회로로 모델링됨 (밀봉 시)

### 중간 확신 (근거 있으나 검증 필요)
- |Z|₀.₁Hz 등급 기준의 아노다이징 적용 타당성 — **6xxx 합금에서만 검증**, 2xxx/7xxx는 미검증
- 봉공 방식별 ARI Gate 기준의 적절성 — 체계적 비교 연구 부족
- 가속시험(336h NSS)-필드 성능 상관 — Piccinini 2023이 낮은 상관을 보고

### 낮은 확신 (탐색적)
- ARI 가중치 및 등급 경계값 — Phase 2에서 반드시 교정
- 비지도 ML의 아노다이징 직접 적용 가능성 — 전용 데이터셋 재검증 필수
- 시간 적분 지표의 아노다이징 적용 유효성 — 단일 연구, 유기 코팅 전용

## 10. 후속 액션 제안

1. **즉시**: 현재 EIS 데이터에서 |Z|₀.₁Hz 히스토그램 작성 + **합금별/봉공별 층화** → 현재 공정 분포 파악
2. **1개월 내**: AHP 기반 초기 ARI 가중치 설정 (전문가 3~5인). "가중치가 틀릴 수 있는 조건" 문서화
3. **3개월 내**: ARI 시범 적용 + 가속시험 병행. **봉공 방식별 Gate 기준 별도 검증**
4. **6개월 내**: Phase 2 상관 분석 완료. ARI 등급 경계값 교정 + 환경별 프로파일 확정
5. **병행**: EIS 자동 측정 시스템 검토 (AutoEIS 포함). 아노다이징 전용 EIS 데이터베이스 구축 시작
