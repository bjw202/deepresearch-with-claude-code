# 아노다이징 피막 강건성 지수화: 통합 분석 및 제안

> 통합일: 2026-03-18
> 기반 자료: eis-methodology.md, quantification-methods.md, alternative-methods.md

---

## 핵심 결론 요약

1. **표준화된 단일 "강건성 지수"는 아직 존재하지 않는다.** 산업 표준(MIL-PRF-8625, QUALANOD)은 개별 항목별 Go/No-Go 판정만 제공하며, 이를 종합한 단일 숫자 지수는 학계에서도 공식 발표된 것이 없다.

2. **EIS 기반 정량 지표가 가장 유망한 기반이다.** 비파괴, 정량적, 피막 구조 정보를 포함하며, 이미 6종의 파생 지표가 연구되어 있다.

3. **단일 시점 측정의 한계를 극복하는 시간 적분 지표가 최신 트렌드다.** Bongiorno et al.(2024)의 ∫(1/|Z|)dt 지표가 단일 시점 |Z|보다 실제 필드 성능과 더 강한 상관관계를 보인다.

4. **제조 현장 적용을 위해서는 계층적(Tiered) 접근이 현실적이다.** 인라인 전수검사 + 오프라인 로트검사 + 정기 가속시험의 3단계 구조.

## 1. 현재 사용 가능한 EIS 기반 지표 6종 비교

| 지표 | 수식 | 장점 | 한계 | 실용성 |
|------|------|------|------|--------|
| **\|Z\|₀.₁Hz** | 0.1Hz에서 임피던스 크기 | 측정 간편, 등급 기준 확립 (>10⁸=우수, <10⁶=불량) | 단일 시점, 피막 구조 무관 | ★★★★★ |
| **PE%** | (Rp,coated - Rp,bare) / Rp,coated × 100 | bare 대비 직관적 비교 | 기준(bare) 시편 필요 | ★★★☆☆ |
| **CDI%** | (Zi - Zf) / Zi \|₀.₁Hz × 100 | 열화 추적에 적합 | 초기값 의존, 2회 이상 측정 필요 | ★★★★☆ |
| **DF%** | Bode plot 면적비 | 코팅 손상 정량화 | 계산 복잡, 자동화 필요 | ★★☆☆☆ |
| **fb (Breakpoint Freq.)** | 위상각 = -45° 주파수 | 단일 값, 열화 민감 | 건전 피막에서 측정 불가(너무 낮음) | ★★★☆☆ |
| **∫(1/\|Z\|)dt** | 저주파 임피던스 역수의 시간 적분 | 이력 통합, 필드 성능 상관 최우수 | 연속 모니터링 필요 | ★★☆☆☆ |

### 판단

**출하 전 1회 검사 용도**로는 **|Z|₀.₁Hz**가 가장 실용적이다. 등가회로 피팅 없이 raw 데이터에서 직접 읽을 수 있어 해석의 주관성을 최소화하며, 이미 문헌에서 등급 기준이 확립되어 있다.

**열화 추적/품질 트렌드 관리**에는 **CDI%**가 적합하다. 동일 조건에서 초기 측정값 대비 변화율을 추적하므로 시편 간 절대값 편차의 영향이 줄어든다.

## 2. 제안: 아노다이징 피막 강건성 지수 (ARI — Anodize Robustness Index)

### 2.1 기본 개념

EIS를 중심으로 하되, 물리 시험을 보완 지표로 결합하는 **2단계 판정 체계**.

```
[Gate 판정] 최소 기준 미달 항목이 하나라도 있으면 → 즉시 FAIL
[지수 산출] 모든 Gate 통과 → ARI 점수 산출 (0~100)
```

### 2.2 Gate 판정 (필수 통과 기준)

| 항목 | 시험법 | 최소 기준 | 근거 |
|------|--------|----------|------|
| 피막 두께 | 와전류 (ASTM B244) | 사양 하한 -10% | MIL-PRF-8625 |
| 봉공 품질 | 어드미턴스 (ISO 2931) | ≤ 20 μS | QUALANOD |
| 접착력 | Cross-cut (ASTM D3359) | ≥ 4B | 산업 관행 |
| 기본 내식성 | |Z|₀.₁Hz | ≥ 10⁶ Ω·cm² | 문헌 합의 (하한) |

### 2.3 ARI 점수 산출

Gate 통과 후, 아래 항목의 가중합으로 0~100 점수를 산출한다.

```
ARI = w₁·S_eis + w₂·S_seal + w₃·S_adhesion + w₄·S_thickness
```

#### 각 항목의 정규화 (0~1 스케일)

**S_eis (전기화학적 장벽성)** — 가중치 권장: 0.45
```
S_eis = [log₁₀(|Z|₀.₁Hz) - 6] / 3
```
- log₁₀(|Z|) = 6 (10⁶) → S = 0 (Gate 하한)
- log₁₀(|Z|) = 9 (10⁹) → S = 1 (최우수)
- 범위 밖은 클램핑 (min 0, max 1)

**S_seal (봉공 품질)** — 가중치 권장: 0.25
```
S_seal = 1 - (Y / 20)
```
- Y = 어드미턴스(μS). 0 μS → S = 1, 20 μS → S = 0
- 산용해 시험(ASTM B680) 데이터가 있으면:
  `S_seal = 1 - (mass_loss / 40)` (40 mg/dm² = AAMA 기준)

**S_adhesion (접착력)** — 가중치 권장: 0.15
```
Cross-cut 등급 매핑:
  5B → 1.0, 4B → 0.6, 3B → 0.3, 2B → 0.1, ≤1B → 0
```
- Pull-off 데이터가 있으면: `S_adhesion = min(pull_off_MPa / target_MPa, 1.0)`

**S_thickness (두께 균일성)** — 가중치 권장: 0.15
```
S_thickness = 1 - |실측두께 - 목표두께| / (목표두께 × 0.2)
```
- 목표 대비 ±20% 이내에서 선형 감점, 20% 초과는 Gate에서 이미 차단

#### ARI 등급

| ARI 점수 | 등급 | 조치 |
|---------|------|------|
| 85~100 | A (우수) | 출하 적합 |
| 70~84 | B (양호) | 출하 적합, 공정 모니터링 |
| 55~69 | C (보통) | 추가 검사 권장, 사용 환경 제한 고려 |
| < 55 | D (부적합) | 출하 부적합 |

### 2.4 환경별 가중치 프로파일

동남아 등 고온고습 환경을 위해 가중치를 조정할 수 있다:

| 항목 | 일반 | 열대기후(동남아) | 해양환경 |
|------|------|----------------|---------|
| S_eis | 0.45 | **0.55** | **0.55** |
| S_seal | 0.25 | **0.25** | 0.20 |
| S_adhesion | 0.15 | 0.10 | 0.15 |
| S_thickness | 0.15 | 0.10 | 0.10 |

동남아 프로파일에서 EIS 가중치를 높이는 이유: 땀(염분) + 고온고습이 전기화학적 부식을 가속하므로, 피막의 전기화학적 장벽 성능이 가장 결정적이다.

## 3. EIS 해석 주관성 해소 방안

현재 EIS 해석의 핵심 문제는 **등가회로 피팅의 비유일성**이다. 같은 스펙트럼을 다른 회로 모델로 피팅하면 다른 파라미터가 나온다.

### 3.1 회피 전략: 피팅 없는 지표 사용

**|Z|₀.₁Hz와 위상각은 등가회로 피팅 없이 raw 데이터에서 직접 읽는다.** 이것이 가장 강력한 주관성 제거 방법이다.

추가로 활용 가능한 피팅-프리(fitting-free) 지표:
- 중주파 위상각 (10 Hz에서의 위상각): -80° 이상이면 양호
- Breakpoint frequency: 자동 추출 가능 (위상각 = -45° 교차점)
- Bode plot 기울기: 이상적 코팅은 -1, 열화 시 기울기 감소

### 3.2 자동화 전략: AutoEIS 등 ML 기반

- **AutoEIS** (JOSS, 2024): K-K 검증 → 진화 알고리즘 회로 생성 → Bayesian 파라미터 추정
- **Bayesian 모델 선택** (npj Materials Degradation, 2024): 시정수 비 > 3일 때 >90% 정확도
- **해석적 ML** (Electrochimica Acta, 2022): 등가회로 예측 + 전역 최적화 파라미터 식별

### 3.3 권장 접근

**단기(즉시 적용):** |Z|₀.₁Hz 기반 ARI 도입. 피팅 불필요.
**중기(6~12개월):** AutoEIS 등으로 Rp, Rb 자동 추출 파이프라인 구축. ARI에 세분화된 EIS 파라미터 반영.
**장기(1~2년):** 공정 파라미터 + EIS + 가속시험 데이터를 통합하는 ML 예측 모델 구축.

## 4. 제조 현장 적용 로드맵

### Phase 1: 기반 구축 (즉시)

```
[인라인]                    [오프라인 로트 검사]
와전류 두께 (전수)           EIS |Z|₀.₁Hz (로트당 3~5개)
어드미턴스 (N개당 1개)       Cross-cut 접착력 (로트당 2~3개)
        ↓                          ↓
    Gate 판정                  ARI 점수 산출
```

- 측정 시간: EIS는 고주파~0.1Hz까지 약 5~10분
- 장비: 포텐시오스탯 1대 + 와전류 게이지 + 어드미턴스 미터

### Phase 2: 상관관계 확립 (3~6개월)

- 다양한 공정 조건(전해액 농도, 온도, 봉공 시간)의 시편에 대해 ARI + 가속시험(염수분무 336h, 인공땀 168h, 85/85 500h) 동시 수행
- ARI 점수와 가속시험 결과 간 회귀 분석
- ARI 등급 경계값 교정

### Phase 3: 지능화 (6~12개월)

- EIS 자동 측정 + 자동 판정 시스템
- 공정 파라미터(전류밀도, 시간, 온도, 봉공조건) → ARI 예측 모델
- SPC 관리도에 ARI 트렌드 표시

## 5. 동남아 환경 특화 고려사항

### 5.1 땀에 의한 피막 박리 메커니즘

1. 땀 속 NaCl(~20 g/L) + 젖산(~15 g/L, pH 4.7)이 피막 표면에 접촉
2. Cl⁻ 이온이 봉공이 불완전한 다공(pore)을 통해 침투
3. 배리어층에 도달하면 국소적 용해(pitting) 개시
4. 열팽창 차이(Al: 23.1 μm/m·K vs Al₂O₃: 8.1 μm/m·K)로 인해 열 사이클 시 미세 크랙 발생
5. 크랙을 통한 추가 침투 → 피막 박리

### 5.2 동남아 대응 ARI 강화

Gate 기준을 강화할 수 있다:

| 항목 | 일반 기준 | 동남아 강화 기준 |
|------|----------|----------------|
| |Z|₀.₁Hz | ≥ 10⁶ Ω·cm² | ≥ 10⁷ Ω·cm² |
| 어드미턴스 | ≤ 20 μS | ≤ 15 μS |
| 추가: 인공땀 침지 | - | 168h 후 ΔE* < 3, 피팅 없음 |
| 추가: 85/85 내습 | - | 500h 후 CDI% < 20% |

### 5.3 인공땀 시험 프로토콜 (ISO 3160-2 기반)

| 성분 | 농도 |
|------|------|
| NaCl | 20 g/L |
| NH₄Cl | 17.5 g/L |
| 아세트산 | 5 g/L |
| d,l-젖산 | 15 g/L |
| pH | 4.7 (NaOH 조정) |

55°C에서 168시간 침지 후 색차(ΔE*), 피팅 밀도, EIS 변화를 평가.

## 6. 불확실성 및 한계

### 높은 확신 (문헌 합의)
- |Z|₀.₁Hz > 10⁸ Ω·cm²이면 우수한 보호 피막이라는 점
- EIS 등가회로 피팅에 주관성이 있다는 점
- 아노다이징 피막은 2-시정수 등가회로로 모델링된다는 점

### 중간 확신 (근거 있으나 검증 필요)
- |Z|₀.₁Hz를 ARI의 핵심 지표로 사용하는 것의 타당성 — 대부분 유기 코팅 연구에서 도출된 기준이며, 아노다이징 피막에 특화된 대규모 연구는 제한적
- 가중치(0.45/0.25/0.15/0.15)의 적절성 — 전문가 판단 기반이며, 실데이터 검증 필요
- 동남아 Gate 강화 기준의 적절성 — 실제 필드 고장 데이터로 교정 필요

### 낮은 확신 (탐색적)
- ARI 등급 경계값(85/70/55) — 현재는 임의 설정, Phase 2에서 반드시 교정해야 함
- ML 기반 예측 모델의 실용 가능 시점 — 충분한 학습 데이터 확보가 전제

## 7. 후속 액션 제안

1. **즉시**: 현재 EIS 측정 데이터에서 |Z|₀.₁Hz 값을 추출하여 히스토그램 작성 → 현재 공정의 분포 파악
2. **1개월 내**: ARI Gate 기준으로 과거 데이터 역추적 → 실제 필드 불량과의 상관관계 확인
3. **3개월 내**: ARI 시범 적용 → 가속시험(염수분무, 인공땀)과 ARI의 상관관계 확립
4. **6개월 내**: ARI 등급 경계값 교정 + 환경별 프로파일 확정
5. **병행**: EIS 자동 측정/판정 시스템 검토 (AutoEIS 또는 상용 소프트웨어)
